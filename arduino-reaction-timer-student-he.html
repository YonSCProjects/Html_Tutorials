Arduino: טיימר תגובה

בנו טיימר תגובה כדי לבדוק כמה מהר אתם יכולים להגיב לאור.

חומרים

  • Arduino Uno או Nano (1)
  • כבל USB (1)
  • Breadboard (1)
  • נורית LED אדומה (1)
  • נורית LED ירוקה (1)
  • נגדים 220 Ω (2)
  • נגד 10 kΩ (1)
  • כפתור לחיצה (1)
  • חוטי מגשר (~8)

הכנה

שלב 1: פתיחת תוכנת Arduino IDE

1. מצאו את אייקון Arduino IDE על שולחן העבודה או בתפריט התחל. 2. לחצו פעמיים כדי לפתוח את Arduino IDE. 3. חברו את ה-Arduino למחשב באמצעות כבל USB. 4. המתינו ש-Windows יזהה את ה-Arduino (אתם עשויים לשמוע צליל חיבור USB).

שלב 2: הגדרת Arduino IDE

1. לחצו על תפריט Tools בחלק העליון. 2. עברו ל-Tools → Board ובחרו Arduino Uno (או Arduino Nano אם אתם משתמשים ב-Nano). 3. עברו ל-Tools → Port ובחרו את פורט COM עם "Arduino" בשם (לדוגמה, COM3, COM5).

  • אם אתם לא רואים שום פורט, נתקו וחברו מחדש את כבל ה-USB, ואז בדקו שוב.

שלב 3: חיווט הרכיבים

לפני שמתחילים: מצאו את מסילת ה-GND על ה-breadboard שלכם (פס כחול או סימני מינוס לאורך הקצה).

שלב 3.1 - חיבור הארקה (GND)

1. חברו חוט מגשר שחור מפין GND של ה-Arduino אל מסילת ה-GND (פס כחול) על ה-breadboard.

שלב 3.2 - חיווט נורית LED אדומה

לנוריות LED יש קוטביות! רגל ארוכה = חיובי (+), רגל קצרה = שלילי (-)

1. הכניסו את נורית ה-LED האדומה ל-breadboard:

  • רגל ארוכה לשורה כלשהי (לדוגמה: שורה 5)
  • רגל קצרה לשורה אחרת, במרחק 2-3 שורות (לדוגמה: שורה 8)

2. חברו את הרגל הקצרה של ה-LED ל-GND:

  • חוט מגשר שחור מאותה שורה של הרגל הקצרה של ה-LED → מסילת GND

3. חברו נגד 220Ω לרגל הארוכה של ה-LED:

  • רגל אחת של הנגד: באותה שורה של הרגל הארוכה של ה-LED
  • רגל שנייה של הנגד: שורה ריקה חדשה

4. חברו את הנגד אל פין 12 של Arduino:

  • חוט מגשר משורת הנגד (זו שלא נוגעת ב-LED) → Arduino Pin 12

שלב 3.3 - חיווט נורית LED ירוקה

חזרו על אותו תהליך עבור נורית ה-LED הירוקה:

1. הכניסו את נורית ה-LED הירוקה לאזור חדש ב-breadboard:

  • רגל ארוכה לשורה (הרחיקו אותה מנורית ה-LED האדומה)
  • רגל קצרה לשורה אחרת

2. חברו את הרגל הקצרה של ה-LED ל-GND:

  • חוט מגשר שחור → מסילת GND

3. חברו נגד 220Ω לרגל הארוכה של ה-LED:

  • רגל אחת: באותה שורה של הרגל הארוכה של ה-LED
  • רגל שנייה: שורה ריקה חדשה

4. חברו את הנגד אל פין 11 של Arduino:

  • חוט מגשר → Arduino Pin 11

שלב 3.4 - חיווט הכפתור

1. הכניסו את כפתור הלחיצה על פני הרווח המרכזי של ה-breadboard:

  • 2 פינים בצד שמאל של הרווח, 2 פינים בצד ימין
  • הכפתור צריך לשבת שטוח

2. חברו צד אחד של הכפתור ל-פין 2 של Arduino:

  • חוט מגשר מפין כלשהו של הכפתור → Arduino Pin 2

3. חברו את הצד המנוגד של הכפתור ל-GND:

  • חוט מגשר מפין הכפתור בצד השני של הרווח המרכזי → מסילת GND

שלב 4: בדיקת החיווט

העלו תוכנית בדיקה כדי לוודא שהכל מחובר נכון.

1. ב-Arduino IDE, מחקו כל קוד קיים בחלון העורך. 2. העתיקו את הקוד למטה והדביקו אותו ב-Arduino IDE: 

void setup() {
  pinMode(12, OUTPUT);
  pinMode(11, OUTPUT);
}

void loop() {
  digitalWrite(12, HIGH);
  digitalWrite(11, HIGH);
  delay(500);
  digitalWrite(12, LOW);
  digitalWrite(11, LOW);
  delay(500);
}

3. לחצו על כפתור Upload (אייקון החץ ימינה בחלק העליון משמאל). 4. המתינו להודעה "Done uploading" בתחתית. 5. הסתכלו על ה-breadboard שלכם.

נקודת ביקורת CP-1: שתי נוריות ה-LED מהבהבות כל 0.5 שניות.

❌ אם לא: בדקו את קוטביות ה-LED (הרגל הארוכה צריכה להתחבר לפין דרך הנגד). ודאו ש-Arduino מחובר לפורט COM הנכון ב-Tools → Port.

שלבי הבנייה

שלב 1 — העלאת קוד השלד

1. ב-Arduino IDE, לחצו על File → New כדי ליצור sketch חדש. 2. לחצו על File → Save ותנו לו את השם ReactionTimer. 3. מחקו את כל הקוד בחלון העורך. 4. העתיקו והדביקו את הקוד הזה: 

// Reaction Timer
const int RED_LED = 12;
const int GREEN_LED = 11;
const int BUTTON = 2;

enum State { IDLE, WAITING, READY, RESULT };
State currentState = IDLE;

unsigned long waitStartTime = 0;
unsigned long randomDelay = 0;
unsigned long readyTime = 0;

void setup() {
  pinMode(RED_LED, OUTPUT);
  pinMode(GREEN_LED, OUTPUT);
  pinMode(BUTTON, INPUT_PULLUP);

  Serial.begin(9600);
  randomSeed(analogRead(A0));

  Serial.println("=== Reaction Timer ===");
  Serial.println("Press button to start!");

  currentState = IDLE;
}

void loop() {
  // We will add code here in the next steps
}

5. לחצו על כפתור Upload (אייקון החץ ימינה). 6. המתינו להודעה "Done uploading". 7. לחצו על Tools → Serial Monitor (או לחצו Ctrl+Shift+M). 8. בחלון ה-Serial Monitor, מצאו את התפריט הנפתח בפינה הימנית התחתונה ובחרו 9600 baud.

נקודת ביקורת CP-2: ה-Serial Monitor מציג: 

=== Reaction Timer ===
Press button to start!

שלב 2 — הוספת מצב IDLE

1. מצאו את השורה שאומרת // We will add code here in the next steps בתוך הפונקציה loop(). 2. מחקו את שורת ההערה הזו. 3. הוסיפו את הקוד הזה בתוך loop(): 

void loop() {
  bool buttonPressed = (digitalRead(BUTTON) == LOW);

  if (currentState == IDLE) {
    digitalWrite(RED_LED, LOW);
    digitalWrite(GREEN_LED, LOW);

    if (buttonPressed) {
      currentState = WAITING;
      digitalWrite(RED_LED, HIGH);
      randomDelay = random(2000, 5000);
      waitStartTime = millis();
      Serial.println("Wait for green...");
    }
  }
}

4. לחצו על כפתור Upload. 5. המתינו להודעה "Done uploading". 6. לחצו על הכפתור הפיזי על ה-breadboard שלכם.

נקודת ביקורת CP-3: לחצו על הכפתור. נורית ה-LED האדומה נדלקת. ה-Serial Monitor מדפיס: 

Wait for green...

❌ אם הכפתור לא עושה כלום: הדפיסו digitalRead(BUTTON) ב-Serial Monitor כדי לבדוק. אם משתמשים ב-INPUT_PULLUP, לחוץ = LOW. בדקו את החיווט.

שלב 3 — הוספת מצב WAITING

1. מצאו את הסוגריים המסולסלים הסוגרים } של בלוק if (currentState == IDLE). 2. מיד אחרי הסוגריים הסוגרים, הוסיפו את הקוד הזה (עדיין בתוך פונקציית loop()): 

  else if (currentState == WAITING) {
    unsigned long elapsed = millis() - waitStartTime;

    if (buttonPressed) {
      Serial.println("Too soon! Try again.");
      currentState = IDLE;
      digitalWrite(RED_LED, LOW);
    }
    else if (elapsed >= randomDelay) {
      currentState = READY;
      digitalWrite(RED_LED, LOW);
      digitalWrite(GREEN_LED, HIGH);
      readyTime = micros();
      Serial.println("GO!");
    }
  }

3. לחצו על כפתור Upload. 4. המתינו להודעה "Done uploading".

בדיקה 1: לחצו על הכפתור, ואז לחצו שוב מיד (בזמן שנורית ה-LED האדומה דולקת).

נקודת ביקורת CP-4a: ה-Serial מדפיס Too soon! Try again. נורית ה-LED האדומה כבה.

בדיקה 2: לחצו על הכפתור, המתינו בלי ללחוץ.

נקודת ביקורת CP-4b: אחרי 2–5 שניות, נורית ה-LED הירוקה נדלקת. ה-Serial מדפיס GO!

שלב 4 — הוספת מצב READY

1. מצאו את הסוגריים המסולסלים הסוגרים } של בלוק else if (currentState == WAITING). 2. מיד אחרי הסוגריים הסוגרים, הוסיפו את הקוד הזה: 

  else if (currentState == READY) {
    if (buttonPressed) {
      unsigned long reactionMicros = micros() - readyTime;
      float reactionMs = reactionMicros / 1000.0;

      Serial.print("Reaction time: ");
      Serial.print(reactionMs, 1);
      Serial.println(" ms");

      currentState = RESULT;
    }
  }

3. לחצו על כפתור Upload. 4. המתינו להודעה "Done uploading". 5. לחצו על הכפתור כדי להתחיל, המתינו לנורית ה-LED הירוקה, לחצו על הכפתור מהר ככל האפשר.

נקודת ביקורת CP-5: ה-Serial מדפיס את זמן התגובה שלכם, לדוגמה: 

Reaction time: 284.7 ms

שלב 5 — הוספת מצב RESULT (השלמת הלולאה)

1. מצאו את הסוגריים המסולסלים הסוגרים } של בלוק else if (currentState == READY). 2. מיד אחרי הסוגריים הסוגרים, הוסיפו את הקוד הזה: 

  else if (currentState == RESULT) {
    digitalWrite(GREEN_LED, LOW);
    delay(2000);
    Serial.println("\nPress button to try again!");
    currentState = IDLE;
  }

3. לחצו על כפתור Upload. 4. המתינו להודעה "Done uploading". 5. בדקו את לולאת המשחק המלאה.

נקודת ביקורת CP-6: אחרי הצגת זמן התגובה, המערכת ממתינה 2 שניות, ואז מדפיסה: 

Press button to try again!

לחיצה על הכפתור שוב → המחזור חוזר על עצמו.

הקוד שלכם הושלם! טיימר התגובה עובד כעת במלואו.

קוד מלא (לעיון)

לחצו להרחבת הקוד המלא 

// Reaction Timer - Complete
const int RED_LED = 12;
const int GREEN_LED = 11;
const int BUTTON = 2;

enum State { IDLE, WAITING, READY, RESULT };
State currentState = IDLE;

unsigned long waitStartTime = 0;
unsigned long randomDelay = 0;
unsigned long readyTime = 0;

void setup() {
  pinMode(RED_LED, OUTPUT);
  pinMode(GREEN_LED, OUTPUT);
  pinMode(BUTTON, INPUT_PULLUP);

  Serial.begin(9600);
  randomSeed(analogRead(A0));

  Serial.println("=== Reaction Timer ===");
  Serial.println("Press button to start!");

  currentState = IDLE;
}

void loop() {
  bool buttonPressed = (digitalRead(BUTTON) == LOW);

  if (currentState == IDLE) {
    digitalWrite(RED_LED, LOW);
    digitalWrite(GREEN_LED, LOW);

    if (buttonPressed) {
      currentState = WAITING;
      digitalWrite(RED_LED, HIGH);
      randomDelay = random(2000, 5000);
      waitStartTime = millis();
      Serial.println("Wait for green...");
    }
  }
  else if (currentState == WAITING) {
    unsigned long elapsed = millis() - waitStartTime;

    if (buttonPressed) {
      Serial.println("Too soon! Try again.");
      currentState = IDLE;
      digitalWrite(RED_LED, LOW);
    }
    else if (elapsed >= randomDelay) {
      currentState = READY;
      digitalWrite(RED_LED, LOW);
      digitalWrite(GREEN_LED, HIGH);
      readyTime = micros();
      Serial.println("GO!");
    }
  }
  else if (currentState == READY) {
    if (buttonPressed) {
      unsigned long reactionMicros = micros() - readyTime;
      float reactionMs = reactionMicros / 1000.0;

      Serial.print("Reaction time: ");
      Serial.print(reactionMs, 1);
      Serial.println(" ms");

      currentState = RESULT;
    }
  }
  else if (currentState == RESULT) {
    digitalWrite(GREEN_LED, LOW);
    delay(2000);
    Serial.println("\nPress button to try again!");
    currentState = IDLE;
  }
}

פתרון בעיות

בעיהפתרון |------|--------| נוריות LED לא נדלקותבדקו קוטביות: רגל ארוכה → נגד → פין. רגל קצרה → GND. הכפתור תמיד מופעל או אף פעם לא מופעלבדקו חיווט. אם משתמשים ב-INPUT_PULLUP, לחוץ = LOW. הדפיסו digitalRead(BUTTON) לניפוי באגים. "Too soon!" בכל פעםחיווט הכפתור הפוך. החליפו את רגלי הכפתור או שנו את הקוד ל-digitalRead(BUTTON) == HIGH. זמן תגובה הוא תמיד 0 או מספר ענקבדקו שהשתמשתם ב-micros() ולא ב-millis(). ודאו ש-readyTime הוא unsigned long. Serial Monitor ריקהגדירו baud ל-9600. בדקו ש-Tools → Port נכון. העלאה נכשלתבדקו ש-Tools → Board נכון. נסו כבל USB אחר.

הרחבות (אופציונלי)

1. מעקב אחר הזמן הטוב ביותר: הוסיפו משתנה bestTime ועדכנו אותו אם הזמן החדש נמוך יותר. הדפיסו אותו אחרי כל ניסיון. 2. ממוצע של 5 ניסיונות: שמרו את 5 הזמנים האחרונים במערך וחשבו את הממוצע. 3. הוספת זמזם: חברו זמזם לפין 10. השמיעו ביפ כשנורית ה-LED הירוקה נדלקת. 4. מצב שני שחקנים: הוסיפו כפתור שני ונורית LED ירוקה שנייה. ראו מי לוחץ ראשון. 5. מצבי קושי: השתמשו בכפתור שני כדי לעבור בין קל (השהיית 3–5 שניות), בינוני (2–4 שניות), קשה (1–2 שניות).

  • ✅ CP-3: לחיצה על כפתור מדליקה נורית LED אדומה ומדפיסה "Wait for green..."

  • ✅ CP-5: לחיצה על כפתור כשירוק מדפיסה זמן תגובה
  • ✅ CP-6: המשחק מתאפס אחרי 2 שניות וחוזר לתחילת הלולאה
