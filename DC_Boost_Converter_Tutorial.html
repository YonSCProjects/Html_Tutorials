תרגיל 1.2: איך זה עובד? העיקרון הפיזיקלי

Boost Converter עובד בעזרת 3 רכיבים עיקריים:

  1. סליל (Inductor) - מאגר אנרגיה בשדה מגנטי
  2. טרנזיסטור מתג (MOSFET) - פותח וסוגר את המעגל מהר מאוד
  3. דיודה (Diode) - מכוונת את הזרם לכיוון אחד בלבד

איך זה עובד בפשטות:

  1. שלב 1 - טעינה: המתג נסגר → זרם עובר דרך הסליל → הסליל אוגר אנרגיה מגנטית
  2. שלב 2 - שחרור: המתג נפתח → הסליל "זורק" את האנרגיה → המתח קופץ למעלה!
  3. חוזר חלילה: זה קורה מאות אלפי פעמים בשנייה (500KHz+)
  שלב 1 - טעינה:                    שלב 2 - שחרור:

      L (סליל)                          L (סליל)
  Vin──►┤XXXX├──┐                   Vin──►┤XXXX├──┐
        │       │                         │       │
        │       │    D (דיודה)            │       │    D (דיודה)
        │       └──►|──── Vout            │       └──►|──── Vout
        │       │                         │       │        (מתח גבוה!)
        │    ┌──┴──┐                      │    ┌──┴──┐
        │    │ SW  │ ← סגור              │    │ SW  │ ← פתוח
        │    │(ON) │                      │    │(OFF)│
        │    └──┬──┘                      │    └──┬──┘
        │       │                         │       │
  GND───┴───────┘                   GND───┴───────┘

  הסליל אוגר אנרגיה             הסליל משחרר = מתח עולה!

תרגיל 2.3: זיהוי פיני המודול

למודול MT3608 יש 4 פינים שצריך לזהות:

  מבט על הפינים (חזית המודול):

    כניסה (INPUT)              יציאה (OUTPUT)
  ┌──────────────────────────────────────────┐
  │                                          │
  │  IN+                            OUT+     │
  │  (●) ←── מתח חיובי    מתח חיובי ──► (●) │
  │       מהסוללה           מוגבר!           │
  │                                          │
  │  IN-                            OUT-     │
  │  (●) ←── GND (אדמה)    GND (אדמה) ──► (●) │
  │                                          │
  │              ┌───────┐                   │
  │              │ (POT) │ ← פוטנציומטר     │
  │              └───────┘   סובב כדי       │
  │                          לכוון מתח       │
  └──────────────────────────────────────────┘

  קוד צבעים מומלץ:
  ● IN+  = חוט אדום (מתח חיובי מסוללה)
  ● IN-  = חוט שחור (אדמה של סוללה)
  ● OUT+ = חוט כתום (מתח מוגבר חיובי)
  ● OUT- = חוט שחור (אדמה יציאה)

תרגיל 3.1: הכנת הברדבורד

  1. קח את הברדבורד והנח אותו לפניך, כשהשורות האנכיות מימין ומשמאל

  3. זהה את השורות הממוספרות באמצע (a-e ו-f-j)

תרגיל 3.2: הנחת המודול על הברדבורד

  1. הנח את מודול MT3608 על הברדבורד, כשכל פין בשורה נפרדת
  2. ודא שפיני IN ופיני OUT לא באותה שורה!
  3. לחץ בעדינות עד שהפינים נכנסים לנקודות
  מיקום המודול על הברדבורד:

  Power Rails

  IN- בשורה 10, צד שמאל (a-e)
  OUT+ בשורה 8, צד ימין (f-j)
  OUT- בשורה 10, צד ימין (f-j)

תרגיל 3.3: חיבור צד הכניסה (Input)

  1. חוט אדום: חבר ממחזיק הסוללה (+) → לשורה 8, טור a (אותה שורה כמו IN+)
  2. חוט שחור: חבר ממחזיק הסוללה (-) → לשורה 10, טור a (אותה שורה כמו IN-)
  3. אופציונלי: חבר קבל 100µF בין שורה 8 לשורה 10 (צד הכניסה) - לסינון

שים לב לקוטביות! הקבל האלקטרוליטי חייב להיות בכיוון הנכון - הרגל הארוכה (+) לשורה 8 (IN+) והרגל הקצרה (-) לשורה 10 (IN-).

  חיבור צד הכניסה:

     └──► ●

תרגיל 3.4: חיבור צד היציאה (Output)

  1. חוט כתום/אדום: חבר מ-OUT+ (שורה 8, צד ימין) → לפס (+) הימני של הברדבורד
  2. חוט שחור: חבר מ-OUT- (שורה 10, צד ימין) → לפס (-) הימני של הברדבורד
  3. אופציונלי: חבר קבל 100µF נוסף בין (+) ל(-) בצד היציאה - לסינון
  חיבור צד היציאה:

              │  MT3608  │     │
          ═══ OUT-]──●  ●  ●    │

תרגיל 3.5: חיבור LED לבדיקה

נחבר LED עם נגד כדי לראות שהמעגל עובד:

  1. נגד 330Ω: חבר רגל אחת לפס (+) הימני, ורגל שנייה לשורה 20
  2. LED: הרגל הארוכה (אנודה, +) לשורה 20. הרגל הקצרה (קתודה, -) לפס (-)

  3. גע בפרוב השחור בקצה (-) של הסוללה
  4. קרא את המתח - צריך להיות בין 3.0V ל-4.2V
  5. רשום את הערך: ___V
  מולטימטר: 3.72V  (ערך טיפוסי לסוללה טעונה באמצע)

  אם רואים 4.0V-4.2V = מלאה
  אם רואים 3.5V-3.8V = באמצע
  אם רואים מתחת ל-3.0V = צריך להיטען!

מה הערך המקסימלי שסוללת 18650 יכולה לתת?

תרגיל 4.3: הפעלה ראשונה!

  1. בדיקה אחרונה: עבור על כל החיבורים ווודא שהכל נכון
  2. הכנס את הסוללה למחזיק (שים לב לכיוון + ו-!)
  3. מה קורה? ה-LED אמור לדלוק (אולי חלש, אולי חזק - תלוי בכיוון הפוטנציומטר)
  4. אם ה-LED לא דולק - נתק מיד את הסוללה ובדוק חיבורים!
  תרחישים אפשריים:

  ✅ LED דולק בהיר    → הכל עובד! מתח היציאה מספיק
  ✅ LED דולק חלש     → עובד, אבל מתח היציאה נמוך - צריך כיוון
  ❌ LED לא דולק      → בדוק: קוטביות LED, חיבורי חוטים, סוללה טעונה?
  ❌ משהו מתחמם      → נתק מיד! יש קצר במעגל!

תרגיל 4.4: כיוון המתח ל-5V

זה השלב הכי חשוב! נכוון את מתח היציאה ל-5V בדיוק:

  1. הנח את פרובי המולטימטר: אדום על OUT+ (או פס +), שחור על OUT- (או פס -)
  2. קרא את המתח הנוכחי על המולטימטר
  3. קח מברג קטן וסובב את הפוטנציומטר (הבורג הקטן על המודול)
  4. סובב ימינה (עם כיוון השעון) → המתח עולה
  5. סובב שמאלה (נגד כיוון השעון) → המתח יורד
  6. כוון ל-5.0V (בין 4.9V ל-5.1V זה מצוין)
  7. הסר את המברג ובדוק שהמתח לא השתנה
  מולטימטר בזמן כיוון:

  סובב שמאלה ←  3.2V ... 3.8V ... 4.5V ... 4.9V → סובב ימינה
                                              ↓
                                           ► 5.00V ◄  ← המטרה!
                                              ↓
                              5.1V ... 5.5V ... 6.0V
                              (לא לעלות יותר מדי!)

לא לעבור 5.5V! אם מתכננים לחבר ל-AMS1117 שמוריד ל-3.3V, מתח גבוה מדי יכול להזיק לרכיבים. 5V זה מושלם.

לאיזה כיוון סובבים את הפוטנציומטר כדי להעלות מתח?

תרגיל 4.5: בדיקת יציבות

נבדוק שהמתח יציב ולא משתנה:

  1. השאר את המולטימטר מחובר ליציאה
  2. המתן 30 שניות - המתח צריך להישאר יציב (±0.05V)
  3. נגע קלות בברדבורד - המתח לא אמור לזוז
  4. כסה את החדר (כבה אור) - המתח לא אמור להשתנות
  5. אם המתח "קופץ" - יש חיבור רופף!
  ✅ יציבות טובה:    5.00V → 30 שניות → 5.00V     (±0.02V)
  ⚠️ קצת לא יציב:   5.00V → 30 שניות → 4.95V     (בדוק חיבורים)
  ❌ לא יציב:        5.00V → 30 שניות → 4.50V     (חיבור רופף!)
✅ עצור! וודא שהתוצאות שלך תקינות:
  • מתח יציאה: 5.0V ± 0.1V
  • LED דולק בהיר
  • שום רכיב לא מתחמם
  • המתח יציב לאורך זמן
