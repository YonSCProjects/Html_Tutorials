בניית מכפיל מתח בעצמך - Charge Pump

הגבר מתח בלי סליל! רק טרנזיסטורים, קבלים, דיודות ונגדים

מה נבנה? מעגל שלוקח 3.7V מסוללה ומכפיל אותו ל-~6V. בלי שום סליל (inductor) ובלי מודולים מוכנים - רק רכיבים בסיסיים שיש לך!

המעגל בנוי משני חלקים:

  1. אוסצילטור (מחולל תנודות) - שני טרנזיסטורים שמייצרים אות מרובע
  2. משאבת מטען (Charge Pump) - קבלים ודיודות שמכפילים את המתח

תרגיל 1.1: הרעיון של משאבת מטען (Charge Pump)

תחשוב על דלי מים:

  1. מלא דלי מים מהברז (גובה 1 מטר)
  2. תעלה את הדלי המלא למדף (גובה 2 מטר)
  3. שפוך את המים - עכשיו הם בגובה 2 מטר!

זה בדיוק מה ש-Charge Pump עושה, אבל עם מתח חשמלי:

  שלב 1: "מילוי הדלי"              שלב 2: "הרמה ושפיכה"
  ═══════════════════               ═══════════════════════

  Vcc (3.7V)                        Vcc (3.7V)
   │                                 │
   │    D1                           │
   ├───►|────┐                       ├───►|────┐
   │         │                       │         │
   │     ────┴────                   │     ────┴────
   │     ║  C3   ║ ← קבל נטען      │     ║  C3   ║ ← קבל טעון
   │     ║ 3.0V  ║    ל-3V          │     ║ 3.0V  ║
   │     ────┬────                   │     ────┬────
   │         │                       │         │
   │         ▼                       │         ▼
   │     GND (0V)                    │     Vcc (3.7V) ← דוחפים למעלה!
   │                                 │         │
   │                                 │   3.0V + 3.7V = 6.7V!
   │                                 │         │
   │                                 │     D2  ▼
   │                                 │    ►|────── Vout ≈ 6.0V!
   │                                 │              │
   │                                 │          ════╧════
   │                                 │          ║  C4   ║
   │                                 │          ╚═══════╝
   │                                 │              │
  GND                               GND           GND

  המתג סגור = קבל נטען             המתג פתוח = קבל מורם
  מ-Vcc דרך דיודה D1               צד התחתון קופץ ל-Vcc

                                    דרך D2 ליציאה → ~6.0V

תרגיל 1.2: למה צריך אוסצילטור?

כדי ש-Charge Pump יעבוד, צריך מתג שנפתח ונסגר מהר מאוד - אלפי פעמים בשנייה!

  • מתג סגור: C3 נטען מ-Vcc (מילוי הדלי)
  • מתג פתוח: C3 מורם ומוסיף מתח (הרמת הדלי)
  • ככל שזה קורה יותר מהר → יותר זרם זמין ביציאה

האוסצילטור שלנו ישתמש בשני טרנזיסטורים שמדליקים ומכבים אחד את השני - נקרא Astable Multivibrator (מולטיוויברטור א-יציב).

  אוסצילטור = מתג שנדלק ונכבה אוטומטית

  מצב 1:                            מצב 2:
  Q1 = ON, Q2 = OFF                 Q1 = OFF, Q2 = ON

    Vcc          Vcc                  Vcc          Vcc
     │            │                    │            │
     R1           R2                   R1           R2
     │            │                    │            │
     ├── LOW      ├── HIGH             ├── HIGH     ├── LOW
     │            │                    │            │
    Q1(ON)       Q2(OFF)              Q1(OFF)      Q2(ON)
     │            │                    │            │
    GND          GND                  GND          GND

           ← מתחלפים אוטומטית! →

  יציאה מ-Q2 collector:
  ┌─────┐     ┌─────┐     ┌─────┐
  │HIGH │     │HIGH │     │HIGH │
  │     │     │     │     │     │
  ┘     └─────┘     └─────┘     └───  ← גל מרובע!

כמה מצבים יש לאוסצילטור Astable Multivibrator?

✅ עצור! וודא שאתה מבין:
  • Charge Pump = קבל שנטען ואז "מורם" כדי להכפיל מתח
  • צריך מתג מהיר (אוסצילטור) שעובד אלפי פעמים בשנייה
  • Astable Multivibrator = שני טרנזיסטורים שמחליפים ביניהם

תרגיל 3.1: סכמה מלאה של האוסצילטור

זו הסכמה של Astable Multivibrator - מעגל שמייצר גל מרובע:

                        Vcc (3.7V)
                           │
              ┌────────────┼────────────┐
              │            │            │
             R1           │           R2
            1kΩ           │          1kΩ
              │            │            │
              ├──── A      │      B ────┤
              │            │            │
              │      ┌─────┘─────┐      │
              │      │           │      │
              │     R3          R4      │
              │    10kΩ        10kΩ     │
              │      │           │      │
     C2       │      │           │      │       C1
  ┌──┤├───────┼──────┘     ┌─────┘      ├───┤├──┐
  │           │            │            │        │
  │           │            │            │        │
  │        ┌──┴──┐      ┌──┴──┐         │        │
  │        │ Q1  │      │ Q2  │         │        │
  │     ┌──┤ NPN │   ┌──┤ NPN │         │        │
  │     │  └──┬──┘   │  └──┬──┘         │        │
  └─────┘     │      └─────┘     ┌──────┘        │
     ▲        │         ▲        │               │
     │        │         │        │               │
  מ-B של Q2   │     מ-B של Q1    │               │
  ל-collector │     ל-collector  │               │
     של Q1    │        של Q2     │               │
              │                  │               │
              └──────────────────┴───────────────┘
                               │
                              GND

   חיבורים צולבים:
   ● C1: מ-collector Q2 (נקודה B) → ל-base Q1
   ● C2: מ-collector Q1 (נקודה A) → ל-base Q2
   ● R3: מ-Vcc → ל-base Q1
   ● R4: מ-Vcc → ל-base Q2

איך זה עובד?

  1. נניח ש-Q1 דולק (ON) ו-Q2 כבוי (OFF)
  2. Collector Q1 = LOW → דרך C2 מוריד את base Q2 → Q2 נשאר OFF
  3. C2 נטען דרך R4 → בסוף base Q2 עולה → Q2 נדלק!
  4. Q2 ON → Collector Q2 = LOW → דרך C1 מוריד את base Q1 → Q1 נכבה!
  5. חוזר לשלב 1 אבל הפוך!

תרגיל 3.2: בנייה על ברדבורד - תזמון איטי (LED מהבהב)

שלב A: תדר נמוך לבדיקה

קודם נבנה עם קבלים גדולים (10µF) כדי שנוכל לראות את ההבהוב בעין:

  1. הנח Q1 ו-Q2 על הברדבורד, עם רווח של 5 שורות ביניהם. ודא שה-Emitter של כל אחד בשורה נפרדת
  2. חבר Emitter של Q1 ו-Q2 ל-GND (פס כחול/שחור)
  3. חבר R1 (1kΩ): מ-Vcc (פס אדום) → לשורה של Collector Q1 (נקודה A)
  4. חבר R2 (1kΩ): מ-Vcc (פס אדום) → לשורה של Collector Q2 (נקודה B)
  5. חבר R3 (10kΩ): מ-Vcc → לשורה של Base Q1
  6. חבר R4 (10kΩ): מ-Vcc → לשורה של Base Q2
  7. חבר C1 (10µF): מ-Collector Q2 (נקודה B) → ל-Base Q1. שים לב לקוטביות! (+) לצד Q2 collector
  8. חבר C2 (10µF): מ-Collector Q1 (נקודה A) → ל-Base Q2. (+) לצד Q1 collector
  9. LED + נגד 1kΩ: חבר LED מ-Collector Q2 (נקודה B) דרך נגד 1kΩ ל-GND. (רגל ארוכה/+ לצד collector)
  סידור על הברדבורד:

   Vcc (+) ═══════════════════════════════════════ (+)
   GND (-) ═══════════════════════════════════════ (-)

         a   b   c   d   e       f   g   h   i   j
    3    ●   ●   ●   ●   ●       ●   ●   ●   ●   ●
    4   [R3 10k from Vcc]        [R4 10k from Vcc]
    5   [to base Q1]             [to base Q2]
    6    ●  [C]  ●   ●   ●       ●   ●  [C]  ●   ●
    7    ●  [B] Q1   ●   ●       ●   ●  [B] Q2   ●
    8    ●  [E]  ●   ●   ●       ●   ●  [E]  ●   ●
    9    ●   │   ●   ●   ●       ●   ●   │   ●   ●
   10    ●  GND  ●   ●   ●       ●   ●  GND  ●   ●

    R1: Vcc → row 6 col b (Q1 collector)
    R2: Vcc → row 6 col h (Q2 collector)
    C2: row 6 col c (Q1 coll) → row 7 col g (Q2 base)
    C1: row 6 col g (Q2 coll) → row 7 col c (Q1 base)

    LED: row 6 col i (Q2 coll) → R7(1kΩ) → GND

    (הערה: מספרי השורות לדוגמה בלבד - התאם למיקום שלך)
        
  אחרי חיבור סוללה:

  LED מהבהב! (נדלק ונכבה לחילופין)

  ✅ עם C1=C2=10µF ו-R3=R4=10kΩ:
     תדר ≈ 1/(0.693 × 2 × 10kΩ × 10µF) ≈ 7.2 Hz
     LED מהבהב ~7 פעמים בשנייה (מהיר אבל נראה!)

  ✅ אם הוא מהבהב - האוסצילטור עובד!
  ❌ אם לא מהבהב - בדוק חיבורי C/B/E של הטרנזיסטורים

תרגיל 3.3: חישוב תדר

הנוסחה לתדר של Astable Multivibrator:

נוסחת תדר (מחזור סימטרי, R3=R4=R, C1=C2=C): f = 1 / (2 × 0.693 × R × C) = 1 / (1.386 × R × C)
R3=R4 C1=C2 תדר שימוש
10kΩ 10µF ~7 Hz בדיקה - רואים הבהוב
10kΩ 100nF ~720 Hz Charge Pump - בינוני
10kΩ 10nF ~7,200 Hz Charge Pump - טוב!
10kΩ 1nF ~72,000 Hz Charge Pump - מצוין!

מה יהיה התדר עם R=10kΩ ו-C=10nF?

✅ עצור! לפני שממשיכים:
  • ה-LED מהבהב? אם כן - האוסצילטור עובד מושלם!
  • אם לא מהבהב: בדוק C/B/E של הטרנזיסטורים, כיוון הקבלים, וחיבורים צולבים

משימה 4: הוספת Charge Pump - הכפלת המתח!

⏱️ 25 דקות

תרגיל 4.1: העלאת התדר

שלב B: עוברים לתדר גבוה

כדי שה-Charge Pump יעבוד טוב, צריך תדר גבוה. נחליף את קבלי התזמון:

  1. נתק את הסוללה!
  2. הוצא את C1 ו-C2 (הקבלים 10µF)
  3. הכנס C1' ו-C2' = 10nF (קרמיים, כתוב עליהם 103) באותם מקומות
  4. הוצא את ה-LED (לא צריך עכשיו)

למה 10nF? עם 10nF האוסצילטור עובד ב-~7,200Hz. בתדר הזה ה-Charge Pump מעביר מטען 7,200 פעם בשנייה - הרבה יותר יעיל!

קבל קרמי = אין כיוון! בניגוד לאלקטרוליטי, לקבל קרמי אין + ו- . אפשר להכניס בכל כיוון.

תרגיל 4.2: הוספת טרנזיסטור באפר (Q3)

Collector של Q2 לא מספיק חזק כדי להזרים מספיק זרם ל-Charge Pump. נוסיף טרנזיסטור שלישי כ-"מגבר כוח":

  למה באפר?

  בלי באפר:                      עם באפר:
  ┌──────────────────┐           ┌──────────────────────────┐
  │ Q2 collector     │           │ Q2 collector             │
  │     │            │           │     │                    │
  │     R2 (1kΩ)     │           │     R5 (10kΩ)           │
  │     │            │           │     │                    │
  │  מוגבל ל-3.7mA  │           │     ▼ Q3 base            │
  │  לא מספיק!      │           │  ┌──┴──┐                 │
  │     │            │           │  │ Q3  │← יכול להעביר   │
  │     ▼            │           │  │ NPN │   עד 100mA!    │
  │  Charge Pump     │           │  └──┬──┘                 │
  │  (לא עובד טוב)  │           │     │ collector          │
  │                  │           │     ▼                    │
  └──────────────────┘           │  Charge Pump             │
                                 │  (עובד מצוין!)           │
                                 └──────────────────────────┘
  1. הנח Q3 (NPN) על הברדבורד, ליד Q2
  2. חבר Emitter Q3 → GND
  3. חבר R5 (10kΩ): מ-Collector Q2 (נקודה B) → ל-Base Q3
  4. חבר R6 (100Ω): מ-Vcc → ל-Collector Q3 (נקודה C)
  Q3 - מגבר באפר:

              Vcc
               │
              R6 (100Ω)    ← pull-up: מושך collector למעלה כש-Q3 OFF
               │
  Q2 coll ── R5(10k) ──► base Q3
                           │
                        ┌──┴──┐
                        │ Q3  │
                        │ NPN │
                        └──┬──┘
                           │ collector = נקודה C (יציאה!)
                           │
                           │ emitter
                           │
                          GND

  כש-Q2 coll = HIGH → Q3 base = HIGH → Q3 ON → נקודה C = LOW (~0V)
  כש-Q2 coll = LOW  → Q3 base = LOW  → Q3 OFF → נקודה C = HIGH (~Vcc)

  (Q3 הופך את האות, וגם מגביר את הכוח!)

תרגיל 4.3: חיבור ה-Charge Pump!

עכשיו מגיע החלק המרגש - נוסיף את הקבל והדיודות שמכפילים את המתח:

  1. חבר D1 (1N4148): Anode ל-Vcc, Cathode לשורה חדשה (נקודה X)
  2. חבר C3 (1µF): צד אחד לנקודה X, צד שני ל-Collector Q3 (נקודה C)
  3. חבר D2 (1N4148): Anode לנקודה X, Cathode לשורה חדשה (נקודה Y = Vout)
  4. חבר C4 (47µF): (+) לנקודה Y (Vout), (-) ל-GND. שים לב לקוטביות!

תרגיל 6.1: שיפור 1 - דיודות שוטקי

אם יש לך דיודות שוטקי (1N5817/1N5819), החלף את D1 ו-D2:

1N4148 (סיליקון רגיל)

  • Vf = 0.7V × 2 = 1.4V הפסד
  • Vout = 2×3.7 − 1.4 = 6.0V

1N5817 (שוטקי)

  • Vf = 0.3V × 2 = 0.6V הפסד
  • Vout = 2×3.7 − 0.6 = 6.8V

דיודות שוטקי גם מהירות יותר, מה שמשפר את היעילות בתדרים גבוהים!

תרגיל 6.2: שיפור 2 - מכפיל כפול (Voltage Tripler)

רוצה עוד יותר מתח? הוסף עוד שלב של C+D!

  Voltage Doubler (מה שבנינו):
  3.7V → [C3 + D1 + D2 + C4] → ~6.0V

  Voltage Tripler (שלב נוסף):
  3.7V → [C3 + D1 + D2] → [C5 + D3 + D4 + C6] → ~9.0V!

  סכמה:

  Vcc──D1──X──D2──Y──D3──Z──D4──W (Vout)
           │       │       │       │
          C3      C5      (אופציונלי)
           │       │
          Q3     Q3 coll
         coll   (אותה נקודה!)

  כל שלב מוסיף עוד ~Vcc − Vf ≈ 3.0V

מתח גבוה מ-6V יכול להזיק לרכיבים רגישים! אל תחבר מתח גבוה ישירות ל-ESP32.
