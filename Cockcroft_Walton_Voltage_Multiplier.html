מגביר מתח DC - Voltage Multiplier

מעגל שלוקח 5V DC מסוללה או USB ומוציא עד 17V DC - רק עם טרנזיסטורים, קבלים ודיודות!

מה נבנה? ממיר DC-to-DC שמקבל 5V DC (מסוללה, USB, או מטען) ומוציא עד ~17V DC - בלי שום שבבים (IC) ובלי סלילים!

איך זה עובד? המעגל בנוי משני חלקים:

  1. אוסצילטור (מנוע פנימי): שני טרנזיסטורים שהופכים את ה-DC לאות מתחלף - זה ה"מנוע" שדוחף מטען דרך הסולם
  2. סולם מתח (קבלים + דיודות): כל שלב בסולם מוסיף עוד ~5V למתח DC ביציאה
  ┌──────────┐     ┌───────────────────────┐     ┌──────────┐
  │  5V DC   │ ──► │  אוסצילטור + סולם מתח  │ ──► │ ~17V DC  │
  │ (כניסה)  │     │   (הקסם קורה בפנים!)   │     │ (יציאה)  │
  └──────────┘     └───────────────────────┘     └──────────┘

במעגל הזה אנחנו מגיעים למתחים של 15V-20V!

  • אל תיגע בחוטים חשופים כשהמעגל דולק
  • השתמש בקבלים עם דירוג מתח (voltage rating) מתאים - לפחות 25V
  • נתק את המתח לפני שינויים במעגל

משימה 1: העיקרון - איך מגבירים מתח DC?

⏱️ 10 דקות

תרגיל 1.1: הרעיון - להפוך DC נמוך ל-DC גבוה

יש לך סוללה של 5V, אבל אתה צריך 17V. מה עושים? בונים סולם מתח!

דמיין שרשרת של דליים על סולם:

  1. דלי ראשון מתמלא מהברז (5V)
  2. דלי ראשון מעביר את המים לדלי שני, שכבר בגובה - עכשיו 10V!
  3. דלי שני מעביר לדלי שלישי, שעוד יותר גבוה - 15V!
  4. כל דלי = שלב (stage) נוסף במעגל

כל "דלי" במעגל בנוי מ:

  • קבל (Capacitor) - אוגר את המטען (ה"מים")
  • דיודה (Diode) - שסתום חד-כיווני שמוודא שהמטען עולה רק למעלה ולא חוזר
  5V DC ──► [אוסצילטור] ──► סולם מתח ──► 17V DC !

  מקור DC                   Stage 1              Stage 2              Stage 3
  (5V)                     (~10V DC)            (~15V DC)           (~17V DC)
                         ┌──────────┐         ┌──────────┐         ┌──────────┐
                         │ +5V      │         │ +5V      │         │ +5V      │
                         │          │         │          │         │          │
  ~~~  ──── C1 ──── D1 ──── C2 ──── D2 ──── C3 ──── D3 ──── C4 ──── D4 ──── Vout
                    │              │              │              │
                   GND            GND            GND            GND
                         │          │         │          │         │          │
                         │  ~10V    │         │  ~15V    │         │  ~20V    │
                         └──────────┘         └──────────┘         └──────────┘

  כל שלב מוסיף עוד ~5V!

אם יש לנו אות AC של 5V שיא (peak), כמה מתח נקבל עם 3 שלבים?

תרגיל 1.2: תפקיד הדיודה - השסתום

הדיודה היא הרכיב הקריטי ביותר בסולם המתח. היא עובדת כמו שסתום חד-כיווני:

  דיודה = שסתום חד-כיווני:

  כיוון מותר (Forward):         כיוון חסום (Reverse):
  ═══════════════════           ═══════════════════════

       זרם ──►                       זרם ──►

    A ──►|── K                    A ──►|── K
        D1                            D1
    (Anode) (Kathode)            (Anode) (Kathode)

    V_A > V_K                    V_A < V_K
    ✅ זרם עובר!                ❌ זרם חסום!
    (ירידת מתח ~0.7V)           (לא עובר שום דבר)


  בסולם המתח:

    Stage 1        Stage 2
       │              │
   C1 charges    C2 charges
       │              │
       ▼              ▼
   ───►|───      ───►|───
      D1            D2
       │              │
       ▼              ▼
   מטען עולה     מטען עולה
   רק למעלה!    עוד למעלה!

  הדיודות מונעות מהמטען לחזור אחורה!

מה קורה כשזרם מנסה לעבור "אחורה" דרך דיודה?

תרגיל 1.3: הטריק - אוסצילטור כ"מנוע פנימי"

סולם מתח צריך משאבה שדוחפת מטען מעלה. הטריק: שני טרנזיסטורים הופכים את ה-DC לאות מתחלף בתוך המעגל. אתה מחבר DC, ובפנים המעגל דואג לכל השאר!

  מה קורה בתוך המעגל:

  5V DC                        אוסצילטור פנימי            סולם מתח
  (כניסה)                      (שני טרנזיסטורים)          (קבלים + דיודות)

  5V ─────────────────  ──►    5V ┌─────┐     ┌───  ──►   17V DC
     │                             │     │     │           (יציאה)
     │  DC מהסוללה                │     │     │
     │                             │     │     │
     │                             │     │     │
  0V ─────────────────         0V ┘     └─────┘

  אתה מחבר DC             האוסצילטור הופך        הקבלים והדיודות
  מסוללה או USB            את זה לאות מתחלף      מעלים את המתח
                           (7100 פעמים בשנייה!)    בחזרה ל-DC גבוה!

  כל פעם שהאות עולה → הקבל נטען
  כל פעם שהאות יורד → המטען נדחף לשלב הבא
  היציאה = DC יציב בערך ~17V!

מה תפקיד האוסצילטור במעגל הזה?

✅ עצור! וודא שאתה מבין: מחברים DC → האוסצילטור הפנימי "שואב" מטען דרך סולם של קבלים ודיודות → מקבלים DC גבוה יותר!

תרגיל 2.2: הכרת הטרנזיסטור NPN

הטרנזיסטור הוא כמו ברז חשמלי - זרם קטן ב-Base פותח זרם גדול מ-Collector ל-Emitter. נשתמש בשניים כדי לבנות אוסצילטור!

  טרנזיסטור NPN - 2N2222 (מבט מלמעלה, צד שטוח לפניך):

      ┌─────────┐
      │  שטוח   │
      │         │
      └──┬─┬─┬──┘
         │ │ │
         E B C

  E = Emitter (פולט)   → מתחבר ל-GND
  B = Base (בסיס)      → "הברז" - שולט על הזרם
  C = Collector (אוגר) → מתחבר דרך נגד ל-Vcc


  איך הטרנזיסטור עובד:
  ══════════════════════

     Vcc (+5V)                    Vcc (+5V)
      │                            │
      Rc (1KΩ)                     Rc (1KΩ)
      │                            │
      ├── C                        ├── C
      │                            │
  B ──┤  (אין זרם ב-B)       B ──┤  (יש זרם ב-B!)
      │                            │
      E                            E
      │                            │
     GND                          GND

  C = HIGH (5V)                 C = LOW (0V)
  טרנזיסטור כבוי ❌             טרנזיסטור דולק ✅

כשטרנזיסטור NPN "דולק" (מקבל זרם ב-Base), מה המתח ב-Collector?

✅ עצור! וודא שיש לך את כל הרכיבים ושאתה מבין את הפינים של הטרנזיסטור (E, B, C).

תרגיל 3.1: מעגל Astable Multivibrator

שני טרנזיסטורים שמדליקים ומכבים אחד את השני - בלי שום שבב (IC)! כל אחד "דוחף" את השני למצב ההפוך, מה שיוצר אות מרובע.

  Astable Multivibrator - תרשים סכמטי:

                Vcc (+5V)                    Vcc (+5V)
                  │                            │
                 Rc1 (1KΩ)                    Rc2 (1KΩ)
                  │                            │
          ┌───── C1 ─────── Ca (10nF) ────── B2 ──┐
          │       │                            │   │
          │      Q1 (2N2222)             Q2 (2N2222)│
          │       │                            │   │
          │      E1                           E2   │
          │       │                            │   │
          │      GND                          GND  │
          │                                        │
          └── Rb1 (10KΩ) ── B1              C2 ────┘
                              │              │
                              │   Cb (10nF)  │
                              └──────────────┘

                                      C2 = OUT
                                  (to voltage ladder)


  פישוט:
  ═══════
  Vcc──Rc1──C1──Ca──B2     Vcc──Rc2──C2──Cb──B1
              │                          │
             Q1                         Q2
              │                          │
             GND                        GND
                                    OUT = C2
  1. הנח את Q1 ו-Q2 על ה-Breadboard (צד שטוח לפניך: E-B-C משמאל לימין)
  2. Q1: חבר E1 (Emitter) → GND
  3. Q1: חבר Rc1 (1KΩ) בין Vcc (+5V) ל-C1 (Collector)
  4. Q2: חבר E2 (Emitter) → GND
  5. Q2: חבר Rc2 (1KΩ) בין Vcc (+5V) ל-C2 (Collector)
  6. חיבור צולב 1: חבר Ca (10nF) בין C1 (Collector של Q1) ל-B2 (Base של Q2)
  7. חיבור צולב 2: חבר Cb (10nF) בין C2 (Collector של Q2) ל-B1 (Base של Q1)
  8. חבר Rb1 (10KΩ) בין Vcc (+5V) ל-B1 (Base של Q1)
  9. חבר Rb2 (10KΩ) בין Vcc (+5V) ל-B2 (Base של Q2)

תדר האוסצילטור:

עם Rb=10KΩ, C=10nF, התדר יהיה בערך 7.1KHz - מושלם לסולם מתח!

נוסחת תדר Astable Multivibrator (סימטרי): f = 1 / (1.4 × Rb × C) = 1 / (1.4 × 10KΩ × 10nF) ≈ 7.1 KHz

מה התדר המשוער של האוסצילטור שלנו (בקירוב)?

תרגיל 3.2: בדיקת האוסצילטור

לפני שמחברים את סולם המתח, נוודא שהאוסצילטור עובד:

  1. חבר את מקור המתח 5V (USB) לפסי + ו-− של ה-Breadboard
  2. חבר LED עם נגד 1KΩ (בסדרה) מ-C2 (Collector של Q2) ל-GND
  3. הדלק את המתח
LED should glow dimly (it's blinking too fast to see individual flashes)

If you have a multimeter on AC mode:
  C2 (Q2 Collector) to GND → should show ~2-3V AC

✅ עובד נכון

  • LED דולק (חלש/עמום)
  • מולטימטר AC מראה 2-3V
  • הטרנזיסטורים לא מתחממים

❌ משהו לא בסדר

  • LED לא דולק בכלל - בדוק חיבורי E/B/C
  • LED דולק חזק ולא מרצד - בדוק קבלי Ca/Cb
  • טרנזיסטור מתחמם - בדוק חיבור E ל-GND

למה ה-LED נראה דולק ברציפות ולא מרצד?

✅ עצור! וודא שהאוסצילטור עובד - LED דולק עמום והטרנזיסטורים לא מתחממים. רק אז המשך!

תרגיל 4.1: הבנת שלב כפול (Stage 1)

שלב 1: מכפיל מתח

השלב הראשון לוקח 5V ומכפיל ל-~10V. הנה מה שקורה צעד אחר צעד:

  שלב 1 - מכפיל מתח (Voltage Doubler):

  OSC OUT (Q2 Collector)
      │
      │         C1 (10μF)              D1 (1N4148)
      ├───────── ┤├+ ──────────────────►|────────┐
      │                                          │
      │                                          │
      │         D2 (1N4148)                      │  Vout1
      ├──────── |◄──────────┐                    │  ~10V
      │                     │                    │
      │                     │    C2 (10μF)       │
      │                     └──── +┤├─ ──────────┤
      │                                          │
     GND                                        GND


  מחצית ראשונה (OUT = LOW):        מחצית שנייה (OUT = HIGH):
  ══════════════════════════        ═══════════════════════════

  OUT = 0V                          OUT = 5V
     │                                 │
     │    C1                           │    C1 (charged!)
     ├──── ┤├+ ──── X                  ├──── ┤├+ ──── D1
     │     charges                     │              │
     │     to ~5V                      │     5V + 4.3V = 9.3V!
     │                                 │              │
     ├──── D2 ──── charges C2          │         C2 charges
     │              to ~4.3V           │         to ~9V
    GND                               GND

למה ~9V ולא 10V? כל דיודה "גונבת" ~0.7V (forward voltage drop). עם 2 דיודות: 10V - 1.4V ≈ 8.6V. בפועל נקבל 8-9V.

כמה מתח כל דיודה 1N4148 "גונבת" כשזרם עובר דרכה?

תרגיל 4.2: בניית השלב הראשון על Breadboard

בנייה!

קבלים אלקטרוליטיים הם קוטביים! חבר את הרגל הארוכה (+) לצד הנכון. חיבור הפוך עלול לגרום לנזק!

  1. נתק את המתח לפני חיבור רכיבים חדשים!
  2. הסר את ה-LED שחיברת לבדיקה (אם חיברת)
  3. חבר C1 (10μF): רגל + → C2 (Collector של Q2) - יציאת האוסצילטור, רגל − → נקודה A
  4. חבר D1 (1N4148): אנודה (ללא פס) → נקודה A, קתודה (פס שחור) → נקודה B
  5. חבר D2 (1N4148): אנודה (ללא פס) → GND, קתודה (פס שחור) → נקודה A
  6. חבר C2 (10μF): רגל + → נקודה B, רגל − → GND
  מבט על ה-Breadboard (שלב 1):

  OSC OUT (Q2 Collector)
       │
       │    C1 (+)         (−)                     D1
       ├────┤ 10μF ├────────●── A ──────────►|──── ● ── B
                            │               (anode→kathode)
                            │
                         D2 │
                     GND ──►|──┘
                       (anode→kathode)

                                                    │
                                              C2 (+)│(−)
                                        B ────┤10μF├──── GND

  נקודה A = חיבור בין C1−, D1 anode, D2 kathode
  נקודה B = חיבור בין D1 kathode, C2+

כיוון הדיודה D2: האנודה מחוברת ל...?

תרגיל 4.3: מדידה ובדיקה

  1. הדלק את המתח (5V USB)
  2. חכה 2-3 שניות שהקבלים ייטענו
  3. מדוד עם מולטימטר (DC mode) את המתח על C2: נקודה B ביחס ל-GND
Multimeter DC reading across C2 (point B to GND):

  Expected: 7V - 9V  (ideally ~8.6V)

  If you measure:
    7-9V    → Great! Stage 1 works!
    4-6V    → Check diode direction (D1 and D2)
    0-2V    → Check C1 connection and oscillator output
    > 10V   → Something is wrong, disconnect!
נוסחת מתח יציאה - שלב 1 (מכפיל): Vout1 = 2 × Vpeak − 2 × Vd = 2 × 5V − 2 × 0.7V = 8.6V

מדדת את המתח על C2. מה המתח שקיבלת?

✅ עצור! וודא שמדדת 7V-9V על C2. אם לא - בדוק כיוון דיודות וחיבורי קבלים. רק אז המשך!

תרגיל 5.1: שלב שני - משלש מתח (Voltage Tripler)

שלב 2: משלש מתח

עכשיו נוסיף עוד קבל ועוד דיודה כדי להגיע ל-~13V!

  הוספת שלב 2 (Tripler):

  שלב 1 (כבר בנוי)                    שלב 2 (חדש!)
  ════════════════                     ══════════════

  OSC OUT
     │
     │    C1          D1               D3 (1N4148)
     ├──── ┤├+ ── A ──►|── B ─────────►|────── C
     │                  │                       │
     │          D2      │    C2                 │   C3 (10μF)

תרגיל 6.1: למה המתח יורד תחת עומס?

נסה לחבר LED עם נגד 1KΩ ליציאת סולם המתח (נקודה D). מה קורה למתח?

  1. מדוד את המתח בנקודה D בלי עומס (ללא LED) - רשום
  2. חבר LED + נגד 1KΩ בין נקודה D ל-GND
  3. מדוד שוב את המתח בנקודה D עם עומס - רשום
  4. השווה בין שתי המדידות
Without load:  ~17V  (no current flowing)
With LED+1K:   ~12-14V  (voltage drops!)

The more current you draw, the more the voltage drops.
This is the main limitation of charge pump circuits.

✅ סולם מתח טוב בשביל:

  • עומסים קלים (מעט זרם)
  • LED בודד או כמה
  • מעגלי לוגיקה (CMOS)
  • חיישנים בזרם נמוך

❌ לא מתאים ל:

  • מנועים (צריכים זרם גבוה)
  • חימום / תאורה חזקה
  • טעינת סוללות
  • כל דבר שצריך > 50mA

סיכום - מה למדנו?

הישגים:

  • ✅ בנית אוסצילטור עם שני טרנזיסטורים (Astable Multivibrator)
  • ✅ בנית סולם מתח עם 3 שלבים (קבלים ודיודות)
  • ✅ הגבהת מתח מ-5V ל-~17V!
  • ✅ הבנת הפסדי דיודות ומגבלות זרם
  • ✅ למדת לשפר ביצועים עם רכיבים טובים יותר

נוסחאות חשובות:

מתח יציאה של N שלבים: Vout = (N+1) × Vpeak − (N+1) × Vd
תדר Astable Multivibrator (סימטרי): f = 1 / (1.4 × Rb × C)

מושגים מרכזיים:

מושג הסבר
DC-to-DC Converter מעגל שמקבל DC בכניסה ומוציא DC שונה (גבוה יותר) ביציאה
Voltage Multiplier סולם של קבלים ודיודות שכל שלב מוסיף עוד Vpeak למתח ה-DC ביציאה
Forward Voltage Drop ~0.7V שכל דיודת סיליקון "גונבת" (0.3V בשוטקי)
Astable Multivibrator שני טרנזיסטורים שמדליקים ומכבים אחד את השני ומייצרים גל מרובע
Ripple / ריפל תנודות במתח היציאה - ככל שהעומס כבד יותר, יש יותר ריפל
