משימה 1: התקנת תוכנות

⏱️ 15-20 דקות

לפני שנתחיל לעבוד עם החומרה, צריך להתקין את התוכנות הנדרשות.

תרגיל 1.1: התקנת Arduino IDE

  1. פתח את הדפדפן ועבור לאתר: https://www.arduino.cc/en/software
  2. לחץ על "Windows Win 10 and newer, 64 bits"
  3. לחץ על "Just Download" (אפשר לדלג על התרומה)
  4. הרץ את קובץ ההתקנה ולחץ Next עד לסיום
  5. פתח את Arduino IDE מתפריט ההתחלה
Arduino IDE ייפתח עם חלון ריק ושורת תפריטים למעלה.
אם נפתח - ההתקנה הצליחה!

תרגיל 1.2: הוספת תמיכה ב-ESP32

  1. ב-Arduino IDE, לחץ על File → Preferences
  2. בשדה "Additional Board Manager URLs" הדבק את הכתובת הזו:
https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  1. לחץ OK
  2. עכשיו לחץ על Tools → Board → Boards Manager
  3. בחיפוש כתוב: esp32
  4. מצא את "esp32 by Espressif Systems" ולחץ Install
  5. המתן להתקנה (עשוי לקחת כמה דקות)
בסיום ההתקנה יופיע כפתור "INSTALLED" ליד החבילה.

איך נבדוק שההתקנה הצליחה?

✅ עצור! לפני שממשיכים, וודא ש:
  • Arduino IDE מותקן ונפתח
  • חבילת ESP32 מותקנת (מופיע "INSTALLED")

משימה 3: העלאת הקוד

⏱️ 15-20 דקות

עכשיו נעלה את הקוד ל-ESP32-CAM שיהפוך אותו למצלמת WiFi.

תרגיל 3.1: הגדרת הלוח ב-Arduino IDE

  1. פתח את Arduino IDE
  2. לחץ על Tools → Board → esp32 → AI Thinker ESP32-CAM
  3. לחץ על Tools → Port → בחר את יציאת ה-COM שמצאת קודם
אם לא מוצא "AI Thinker ESP32-CAM", חפש "ESP32 Wrover Module" - גם זה יעבוד.

תרגיל 3.2: העתקת הקוד

  1. ב-Arduino IDE, לחץ File → New Sketch
  2. מחק את כל הטקסט שבקובץ
  3. העתק את הקוד הבא והדבק בחלון:
/*
 * ESP32-CAM WiFi Camera
 * מצלמת WiFi לשידור לטלפון
 */

#include "esp_camera.h"
#include <WiFi.h>
#include "esp_http_server.h"
#include "soc/soc.h"
#include "soc/rtc_cntl_reg.h"

// ========================================
// שנה את הפרטים האלה לרשת ה-WiFi שלך!
// ========================================
const char* ssid = "YOUR_WIFI_NAME";       // שם הרשת שלך
const char* password = "YOUR_WIFI_PASSWORD"; // סיסמת הרשת

// הגדרות מצלמה (AI Thinker ESP32-CAM)
#define PWDN_GPIO_NUM     32
#define RESET_GPIO_NUM    -1
#define XCLK_GPIO_NUM      0
#define SIOD_GPIO_NUM     26
#define SIOC_GPIO_NUM     27
#define Y9_GPIO_NUM       35
#define Y8_GPIO_NUM       34
#define Y7_GPIO_NUM       39
#define Y6_GPIO_NUM       36
#define Y5_GPIO_NUM       21
#define Y4_GPIO_NUM       19
#define Y3_GPIO_NUM       18
#define Y2_GPIO_NUM        5
#define VSYNC_GPIO_NUM    25
#define HREF_GPIO_NUM     23
#define PCLK_GPIO_NUM     22
#define FLASH_GPIO_NUM     4

httpd_handle_t stream_httpd = NULL;
httpd_handle_t camera_httpd = NULL;

#define PART_BOUNDARY "123456789000000000000987654321"
static const char* _STREAM_CONTENT_TYPE = "multipart/x-mixed-replace;boundary=" PART_BOUNDARY;
static const char* _STREAM_BOUNDARY = "\r\n--" PART_BOUNDARY "\r\n";
static const char* _STREAM_PART = "Content-Type: image/jpeg\r\nContent-Length: %u\r\n\r\n";

// דף האינטרנט
static const char PROGMEM INDEX_HTML[] = R"rawliteral(
<!DOCTYPE html>
<html>
<head>
    <title>ESP32-CAM</title>
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <style>
        body { font-family: Arial; background: #1a1a2e; color: #eee;
               margin: 0; padding: 20px; text-align: center; }
        h1 { color: #00d4ff; }
        #stream { width: 100%; max-width: 640px; border: 3px solid #00d4ff; border-radius: 10px; }
        .btn { background: #00d4ff; color: #1a1a2e; border: none; padding: 15px 30px;
               font-size: 18px; border-radius: 10px; cursor: pointer; margin: 10px; }
        .btn:hover { background: #00a8cc; }
    </style>
</head>
<body>
    <h1>ESP32-CAM</h1>
    <img id="stream" src="">
    <br><br>
    <button class="btn" onclick="startStream()">Start</button>
    <button class="btn" onclick="stopStream()">Stop</button>
    <button class="btn" onclick="toggleFlash()">Flash</button>
    <script>
        var streaming = false;
        var flashOn = false;
        function startStream() {
            document.getElementById('stream').src = window.location.origin.replace(':80','') + ':81/stream';
            streaming = true;
        }
        function stopStream() {
            document.getElementById('stream').src = '';
            streaming = false;
        }
        function toggleFlash() {
            flashOn = !flashOn;
            fetch('/flash?state=' + (flashOn ? '1' : '0'));
        }
        setTimeout(startStream, 1000);
    </script>
</body>
</html>
)rawliteral";

static esp_err_t index_handler(httpd_req_t *req) {
    httpd_resp_set_type(req, "text/html");
    return httpd_resp_send(req, (const char *)INDEX_HTML, strlen(INDEX_HTML));
}

static esp_err_t stream_handler(httpd_req_t *req) {
    camera_fb_t *fb = NULL;
    esp_err_t res = ESP_OK;
    char part_buf[64];

    res = httpd_resp_set_type(req, _STREAM_CONTENT_TYPE);
    httpd_resp_set_hdr(req, "Access-Control-Allow-Origin", "*");

    while (true) {
        fb = esp_camera_fb_get();
        if (!fb) { res = ESP_FAIL; break; }

        res = httpd_resp_send_chunk(req, _STREAM_BOUNDARY, strlen(_STREAM_BOUNDARY));
        if (res == ESP_OK) {
            size_t hlen = snprintf(part_buf, 64, _STREAM_PART, fb->len);
            res = httpd_resp_send_chunk(req, part_buf, hlen);
        }
        if (res == ESP_OK) {
            res = httpd_resp_send_chunk(req, (const char *)fb->buf, fb->len);
        }
        esp_camera_fb_return(fb);
        if (res != ESP_OK) break;
    }
    return res;
}

static esp_err_t flash_handler(httpd_req_t *req) {
    char buf[32];
    if (httpd_req_get_url_query_str(req, buf, 32) == ESP_OK) {
        char param[8];
        if (httpd_query_key_value(buf, "state", param, 8) == ESP_OK) {
            digitalWrite(FLASH_GPIO_NUM, atoi(param) ? HIGH : LOW);
        }
    }
    httpd_resp_send(req, "OK", 2);
    return ESP_OK;
}

void startCameraServer() {
    httpd_config_t config = HTTPD_DEFAULT_CONFIG();

    httpd_uri_t index_uri = { .uri = "/", .method = HTTP_GET, .handler = index_handler };
    httpd_uri_t flash_uri = { .uri = "/flash", .method = HTTP_GET, .handler = flash_handler };
    httpd_uri_t stream_uri = { .uri = "/stream", .method = HTTP_GET, .handler = stream_handler };

    if (httpd_start(&camera_httpd, &config) == ESP_OK) {
        httpd_register_uri_handler(camera_httpd, &index_uri);
        httpd_register_uri_handler(camera_httpd, &flash_uri);
    }

    config.server_port = 81;
    config.ctrl_port = 32769;
    if (httpd_start(&stream_httpd, &config) == ESP_OK) {
        httpd_register_uri_handler(stream_httpd, &stream_uri);
    }
}

void setup() {
    WRITE_PERI_REG(RTC_CNTL_BROWN_OUT_REG, 0);
    Serial.begin(115200);
    pinMode(FLASH_GPIO_NUM, OUTPUT);

    camera_config_t config;
    config.ledc_channel = LEDC_CHANNEL_0;
    config.ledc_timer = LEDC_TIMER_0;
    config.pin_d0 = Y2_GPIO_NUM; config.pin_d1 = Y3_GPIO_NUM;
    config.pin_d2 = Y4_GPIO_NUM; config.pin_d3 = Y5_GPIO_NUM;
    config.pin_d4 = Y6_GPIO_NUM; config.pin_d5 = Y7_GPIO_NUM;
    config.pin_d6 = Y8_GPIO_NUM; config.pin_d7 = Y9_GPIO_NUM;
    config.pin_xclk = XCLK_GPIO_NUM; config.pin_pclk = PCLK_GPIO_NUM;
    config.pin_vsync = VSYNC_GPIO_NUM; config.pin_href = HREF_GPIO_NUM;
    config.pin_sccb_sda = SIOD_GPIO_NUM; config.pin_sccb_scl = SIOC_GPIO_NUM;
    config.pin_pwdn = PWDN_GPIO_NUM; config.pin_reset = RESET_GPIO_NUM;
    config.xclk_freq_hz = 20000000;
    config.pixel_format = PIXFORMAT_JPEG;
    config.frame_size = FRAMESIZE_VGA;
    config.jpeg_quality = 12;
    config.fb_count = 2;
    config.grab_mode = CAMERA_GRAB_LATEST;
    config.fb_location = CAMERA_FB_IN_PSRAM;

    if (esp_camera_init(&config) != ESP_OK) {
        Serial.println("Camera init failed!");
        return;
    }
    Serial.println("Camera OK!");

    WiFi.begin(ssid, password);
    Serial.print("Connecting to WiFi");
    while (WiFi.status() != WL_CONNECTED) {
        delay(500);
        Serial.print(".");
    }
    Serial.println("\n\n=========================");
    Serial.print("Camera Ready! Go to: http://");
    Serial.println(WiFi.localIP());
    Serial.println("=========================\n");

    startCameraServer();
}

void loop() {
    delay(10000);
}

תרגיל 3.3: שינוי פרטי ה-WiFi

זה השלב הכי חשוב! בלי לשנות את הפרטים, המצלמה לא תתחבר לרשת שלך.
  1. מצא את השורות האלה בקוד (קרוב לתחילה):
const char* ssid = "YOUR_WIFI_NAME";       // שם הרשת שלך
const char* password = "YOUR_WIFI_PASSWORD"; // סיסמת הרשת
  1. שנה את YOUR_WIFI_NAME לשם הרשת שלך (בדיוק כמו שזה מופיע בטלפון)
  2. שנה את YOUR_WIFI_PASSWORD לסיסמת הרשת שלך
  3. חשוב: השאר את הגרשיים (" ") סביב השם והסיסמה!
דוגמה:
אם הרשת שלך נקראת "BezeqHome" והסיסמה היא "abc12345", תכתוב: 
const char* ssid = "BezeqHome";
const char* password = "abc12345";
  • שם הרשת והסיסמה רגישים לאותיות גדולות/קטנות!
  • ה-ESP32 תומך רק ברשתות 2.4GHz (לא 5GHz)
  • אם יש לך רשת עם שם באנגלית ובעברית, השתמש בשם האנגלי

⏱️ 5 דקות
מתי להשתמש במצב הזה?
אם אין לך רשת WiFi זמינה, או שאתה רוצה להשתמש במצלמה בחוץ/בשטח - המצלמה יכולה ליצור רשת WiFi משלה!

תרגיל 3B.1: הפעלת מצב Access Point

אם אתה רוצה להשתמש במצב Access Point (המצלמה יוצרת רשת משלה), שנה את הקוד כך:

  1. מצא את השורה הזו בתחילת הקוד:
#define USE_ACCESS_POINT false
  1. שנה אותה ל:
#define USE_ACCESS_POINT true
  1. אפשר גם לשנות את שם הרשת והסיסמה שהמצלמה תיצור:
const char* ap_ssid = "ESP32-CAM";      // שם הרשת שתיווצר
const char* ap_password = "12345678";    // סיסמה (מינימום 8 תווים)
שים לב:
  • הסיסמה חייבת להיות לפחות 8 תווים
  • אם רוצים רשת פתוחה (בלי סיסמה), כתוב: const char* ap_password = "";
  • במצב AP הכתובת תמיד תהיה: 192.168.4.1

תרגיל 3B.2: התחברות במצב Access Point

אחרי העלאת הקוד במצב Access Point:

  1. פתח את הגדרות ה-WiFi בטלפון
  2. חפש רשת בשם "ESP32-CAM" (או השם שבחרת)

משימה 5: צפייה מהטלפון

⏱️ 5-10 דקות

עכשיו החלק המרגש - צפייה בשידור מהטלפון!

תרגיל 5.1: חיבור הטלפון לאותה רשת

  1. פתח את הגדרות בטלפון
  2. לחץ על WiFi
  3. וודא שאתה מחובר לאותה רשת שהמצלמה מחוברת אליה
חשוב מאוד! הטלפון חייב להיות מחובר לאותה רשת WiFi כמו המצלמה. אם יש לך שתי רשתות (2.4GHz ו-5GHz), וודא שהטלפון מחובר לרשת ה-2.4GHz.

תרגיל 5.2: צפייה בדפדפן הטלפון

השיטה הכי פשוטה - פשוט לפתוח בדפדפן:

  1. פתח את Chrome או Firefox בטלפון
  2. הקלד את כתובת ה-IP של המצלמה (למשל: 192.168.1.105)
  3. לחץ על Start אם הסטרים לא התחיל אוטומטית
  4. סובב את הטלפון לרוחב לצפייה טובה יותר
בטלפון אמור להופיע:
- וידאו חי מהמצלמה
- כפתורים: Start, Stop, Flash

תרגיל 5.3: אפליקציית IP Camera (אופציונלי)

לחוויית צפייה טובה יותר, אפשר להשתמש באפליקציה ייעודית:

  1. פתח את Google Play Store
  2. חפש: "IP Webcam Viewer" או "tinyCam Monitor"
  3. התקן את האפליקציה
  4. פתח את האפליקציה והוסף מצלמה חדשה
  5. הכנס את כתובת ה-IP של המצלמה
  6. הכנס את נתיב הסטרים: /stream או :81/stream
כתובת סטרים מלאה:
http://192.168.1.105:81/stream
(שנה את ה-IP לכתובת שלך)

תרגיל 5.4: צפייה ב-VLC

אפשר גם להשתמש ב-VLC Player:

  1. התקן VLC for Android מ-Play Store
  2. פתח VLC ולחץ על More (שלוש נקודות)
  3. בחר Stream
  4. הכנס את הכתובת: http://192.168.1.105:81/stream
  5. לחץ על החץ לנגן
✅ מזל טוב! אם אתה רואה וידאו חי בטלפון - הפרויקט הושלם בהצלחה!

למדת: חיבור טלפון לשידור וצפייה בוידאו חי

משימת בונוס: התאמות והרחבות

⏱️ לבחירתך

בונוס 1: שינוי רזולוציית הוידאו

אפשר לשנות את איכות הוידאו בקוד. מצא את השורה:

config.frame_size = FRAMESIZE_VGA;

ושנה אותה לאחת מהאפשרויות:

ערך רזולוציה מתאים ל-
FRAMESIZE_QVGA 320x240 חיבור איטי, רשת עמוסה
FRAMESIZE_VGA 640x480 איזון טוב (ברירת מחדל)
FRAMESIZE_SVGA 800x600 איכות טובה יותר
FRAMESIZE_XGA 1024x768 איכות גבוהה
FRAMESIZE_SXGA 1280x1024 איכות גבוהה מאוד
רזולוציה גבוהה יותר = איכות טובה יותר, אבל גם יותר עיכוב (lag) ויותר עומס על הרשת.

בונוס 2: שמירת תמונות

אפשר לשמור תמונות מהמצלמה על ידי גישה לכתובת:

http://192.168.1.105/capture

זה יוריד תמונת JPEG מהמצלמה!

בונוס 3: רעיונות לפרויקטים

  • מצלמת אבטחה: הנח את המצלמה בכניסה לחדר
  • מעקב אחרי חיות מחמד: צפה בכלב/חתול כשאתה לא בבית
  • מצלמת צמחים: צלם time-lapse של צמח שגדל
  • מצלמה לרכב שלט רחוק: הוסף למכונית RC וצפה מרחוק

🔧 פתרון בעיות

בעיה: "Camera init failed"

  • נתק את הלוח מהחשמל
  • בדוק שהסרט הכתום של המצלמה מחובר היטב
  • הרם את התפס השחור, הכנס את הסרט, וסגור את התפס
  • חבר מחדש וריסטרט

בעיה: WiFi לא מתחבר

  • בדוק שם רשת וסיסמה - רגישים לאותיות גדולות/קטנות!
  • ודא שזו רשת 2.4GHz (ESP32 לא תומך ב-5GHz)
  • קרב את הלוח לראוטר
  • נסה להפעיל נקודת גישה (hotspot) מהטלפון לבדיקה

בעיה: הדפדפן לא מציג כלום

  • וודא שהמחשב/טלפון באותה רשת WiFi
  • נסה לרענן את הדף (F5)
  • נסה דפדפן אחר
  • בדוק שה-IP נכון ב-Serial Monitor

בעיה: וידאו קופץ או איטי

  • הורד את הרזולוציה בקוד
  • קרב את המצלמה לראוטר
  • הפחת מכשירים אחרים ברשת
  • השתמש ברשת 5GHz לשאר המכשירים
