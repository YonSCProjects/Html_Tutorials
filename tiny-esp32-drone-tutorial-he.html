בנה רחפן קוואדקופטר נשלט מהסמארטפון שלך

מה נבנה

רחפן מיני קוואדקופטר נשלט מהסמארטפון שלך דרך Wi-Fi. הוא משתמש ב:

  • ESP32-C3 SuperMini - המוח (מריץ קוד ויוצר נקודת Wi-Fi)
  • MPU6050 - חיישן תנועה (מזהה הטיה לייצוב אוטומטי)
  • בקרי L9110S - בקרי מנועים (2 בקרים ל-4 מנועים)
  • ממיר MT3608 - מעלה מתח מ-3.7V ל-5V
  • סוללת LiPo - מקור כוח 600mAh (4-6 דקות טיסה)
⏱️ זמן בנייה כולל: 4-6 שעות

(מומלץ לפרוס על 2-3 מפגשים)

משימה 1: בניית השלדה

⏱️ 30 דקות

השלדה עשויה משני מקלות ארטיק מודבקים בצורת +.

תרגיל 1.1: סימון המרכזים

  1. קח את שני מקלות הארטיק (12 ס"מ כל אחד)
  2. מדוד 6 ס"מ מכל קצה - זה המרכז
  3. סמן את נקודת המרכז על כל מקל בעיפרון
מדידת מקל ארטיק:
←────── 12cm total ──────→

←─ 6cm ─→│←─ 6cm ─→
         ✕
     center mark

מה האורך הכולל של כל מקל ארטיק?

תרגיל 1.2: הדבקת הצלב

  1. הנח מקל אחד שטוח על השולחן
  2. הנח את המקל השני בניצב (90°) מעליו
  3. יישר את סימני המרכז בדיוק
  4. השתמש בפינת נייר לבדיקת זווית 90°
  5. מרח דבק חם בנקודת החיבור
  6. החזק 30 שניות עד שהדבק מתקשה
  7. המתן 10 דקות להתייבשות
הרכבת השלדה (מבט מלמעלה):
              F (Front)
                 │
                 │
    L ───────────╬─────────── R
   (Left)        │        (Right)
                 │
                 │
              B (Back)

╬ = נקודת הדבקה (מרכז)
פרישה כוללת: ~12 ס"מ מקצה לקצה
השתמש במינימום דבק - טיפה בגודל אפונה מספיקה. יותר מדי דבק מוסיף משקל!

תרגיל 1.3: סימון הזרועות

  1. בחר איזו זרוע תהיה "קדימה" (לעברך)
  2. כתוב F, R, B, L על כל זרוע בטוש
  3. קדימה וימין יקבלו מנועי CW (עם כיוון השעון)
  4. אחורה ושמאל יקבלו מנועי CCW (נגד כיוון השעון)
תמיד שמור על אותו כיוון! הרחפן לא יתאזן אם תערבב את הזרועות.

תרגיל 1.4: קידוח חורי המנועים

  1. מדוד 1 ס"מ פנימה מקצה כל זרוע
  2. סמן את נקודות הקידוח
  3. קדח חורים בקוטר 7mm, לעומק כ-9mm
  4. אל תקדח עד הסוף!
  5. נקה שאריות עץ עם אוויר או נפיחה
מיקומי חורי המנועים:
                 ⊗ (Motor 1)
                 ↑
                1cm from tip
                 │
     ⊗ ──────────╬────────── ⊗
  (Motor 4)      │       (Motor 2)
                 │
                 ⊗ (Motor 3)

⊗ = קדח חור 7mm, עומק 9mm

לאיזה עומק צריך לקדח כל חור?

✅ עצור! בדוק את השלדה שלך:
  • הזרועות בדיוק בזווית 90°
  • כל 4 החורים נקדחו לעומק 9mm
  • הזרועות מסומנות F, R, B, L
  • השלדה יציבה (לא רופפת)

משימה 2: התקנת המנועים

⏱️ 25 דקות

התקן את 4 המנועים עם כיווני סיבוב נכונים.

תרגיל 2.1: הבנת כיווני סיבוב

קוואדקופטרים צריכים מנועים שמסתובבים בכיוונים ספציפיים לאיזון:

תבנית סיבוב המנועים:
         Motor 1 (Front)
              ⊕
              ↻ CW (Clockwise)
              │
              │
Motor 4  ─────┼───── Motor 2
   ⊕          │          ⊕
 ↺ CCW        │        ↻ CW
              │
              ⊕
         Motor 3 (Back)
           ↺ CCW

Front + Right = CW (↻) עם כיוון השעון
Back + Left = CCW (↺) נגד כיוון השעון

אילו מנועים מסתובבים עם כיוון השעון (CW)?

תרגיל 2.2: הכנסת המנועים

  1. קח מנוע 8520 (8.5×20mm) אחד
  2. הכנס אותו לחור כשהציר מצביע למעלה
  3. דחוף עד שגוף המנוע בחצי החור
  4. המנוע צריך להיכנס הדוק אבל לא בכוח
  5. חזור על זה לכל 4 המנועים
התקנת מנוע (מבט צד):
             ↑ Shaft up (for propeller)
┌────────────┼────────────┐
│            ╔═╗          │  ← Frame surface
│            ║ ║          │
│            ║ ║ ← Motor  │
│   Glue → ◯╚═╝◯         │     halfway in
│            hole         │
└─────────────────────────┘

◯ = מרח טבעת דקה של דבק חם
אל תמרח דבק על ציר המנוע או המיסבים! רק דבק סביב השפה שבה המנוע פוגש את העץ.

תרגיל 2.3: איטום בדבק

  1. מרח טבעת דקה של דבק חם סביב שפת כל מנוע
  2. וודא שהדבק נוגע גם בגוף המנוע וגם בעץ
  3. שמור דבק הרחק מהציר
  4. המתן 24 שעות להתייבשות מלאה לחוזק מקסימלי
אפשר להמשיך עם שלבים אחרים בזמן שהדבק מתייבש - רק תהיה עדין עם השלדה.
✅ עצור! סובב כל ציר מנוע עם האצבע:
  • כל 4 הצירים צריכים להסתובב בחופשיות
  • אין שפשוף או התנגדות
  • המנועים לא מתנועעים בחורים שלהם

משימה 3: כיוון ממיר המתח

⏱️ 15 דקות

ממיר MT3608 מעלה 3.7V מהסוללה ל-5V עבור האלקטרוניקה.

תרגיל 3.1: הכנת הבדיקה

  1. הגדר את המולטימטר למתח DC (V⎓)
  2. חבר גששן אדום ל-MT3608 OUT+
  3. חבר גששן שחור ל-MT3608 OUT-
  4. חבר אדום מהסוללה ל-MT3608 IN+
  5. חבר שחור מהסוללה ל-MT3608 IN-
הגדרת כיוון MT3608:
Battery (3.7V)         Multimeter
┌─────────┐           ┌──────────┐
│  +  -   │           │  ?.??V   │
└──┬──┬───┘           └──┬───┬───┘
   │  │                  │   │
 RED BLACK             RED BLACK
   │  │                  │   │
   ▼  ▼                  ▼   ▼
┌─────────────────────────────┐
│ IN+  IN-       OUT+  OUT-   │
│                             │
│       MT3608 Booster        │
│                             │
│         ╔═╗ ← Adjustment    │
│         ║▲║    screw        │
│         ╚═╝                 │
└─────────────────────────────┘

סובב: ↻ = מתח גבוה יותר
      ↺ = מתח נמוך יותר

תרגיל 3.2: כיוון ל-5.0V

  1. המולטימטר מציג את מתח הפלט הנוכחי
  2. השתמש במברג קטן על בורג הכיוון
  3. סובב עם כיוון השעון (↻) להעלאת מתח
  4. סובב נגד כיוון השעון (↺) להורדת מתח
  5. כוון עד שהמולטימטר מראה 5.0V (±0.1V בסדר)
  6. נתק את הסוללה כשסיימת
Multimeter reading: 5.00V
(מ-4.9V עד 5.1V זה מקובל)

איזה מתח צריך MT3608 להוציא?

לעולם אל תעלה מעל 5.5V! מתח גבוה יותר יכול להזיק ל-ESP32.
✅ עצור! וודא שפלט MT3608 יציב על 5.0V לפני שממשיכים.

תרגיל 4.1: פריסת הרכיבים

למד לאן הולך כל רכיב:

פריסת רכיבים (מבט מלמעלה):

תרגיל 4.2: התקנת ESP32

  1. הדבק סרט דו-צדדי לתחתית ESP32
  2. הנח במרכז השלדה
  3. יציאת USB צריכה לפנות החוצה (לתכנות קל)
  4. לחץ חזק למשך 10 שניות

תרגיל 4.3: התקנת MPU6050 (חשוב!)

  1. השתמש בסרט ספוג (לא סרט רגיל!)
  2. הדבק סרט ספוג לתחתית MPU6050
  3. הנח מאחורי ESP32
  4. לחץ בעדינות (אל תמעך את הספוג)
חובה להשתמש בסרט ספוג ל-MPU6050! הספוג סופג רעידות מנועים. בלעדיו, קריאות החיישן יהיו רועשות והרחפן לא יתייצב.

למה חייבים להשתמש בסרט ספוג ל-MPU6050?

תרגיל 4.4: התקנת בקרי L9110S

  1. L9110S #1: התקן על זרוע ימין (שולט במנועי קדימה + ימין)
  2. L9110S #2: התקן על זרוע שמאל (שולט במנועי אחורה + שמאל)
  3. השתמש בסרט דו-צדדי
  4. מקם כך שהפינים נגישים לחיווט

תרגיל 4.5: התקנת MT3608 וסוללה

  1. MT3608: הדבק מתחת למרכז השלדה (צד תחתון)
  2. מקם כך שבורג הכיוון נגיש
  3. סוללה: חבר רצועת סקוטש לתחתית השלדה
  4. הסוללה נכנסת לסקוטש (חייבת להיות ניתנת להסרה!)

משימה 5: חיווט חשמל

⏱️ 45 דקות

חבר את כל חוטי החשמל. השתמש באדום ל-+, שחור ל-GND.

תרגיל 5.1: סקירת חלוקת חשמל

זרימת חשמל מלאה:
Battery (3.7V)
    │
    ├── RED ──→ MT3608 IN+
    └── BLACK → MT3608 IN- (GND)
                    │
                    ├── OUT+ (5V) ──┬──→ ESP32 VIN
                    │               ├──→ L9110S #1 VCC
                    │               └──→ L9110S #2 VCC
                    │
                    └── OUT- (GND) ─┬──→ ESP32 GND
                                    ├──→ L9110S #1 GND
                                    ├──→ L9110S #2 GND
                                    └──→ MPU6050 GND

ESP32 3.3V ──→ MPU6050 VCC (החיישן צריך 3.3V, לא 5V!)

תרגיל 5.2: סוללה ל-MT3608

  1. הלחם חוט אדום ממחבר הסוללה ל-MT3608 IN+
  2. הלחם חוט שחור ממחבר הסוללה ל-MT3608 IN-
תמיד בדוק קוטביות! אדום ל-+, שחור ל-. קוטביות הפוכה תזיק לרכיבים.

תרגיל 5.3: MT3608 ל-ESP32

  1. הלחם חוט אדום מ-MT3608 OUT+ לפין VIN של ESP32
  2. הלחם חוט שחור מ-MT3608 OUT- לפין GND של ESP32

תרגיל 5.4: MT3608 לבקרי L9110S

  1. MT3608 OUT+ → L9110S #1 VCC (השתמש במפצל או חיבור הלחמה)
  2. MT3608 OUT+ → L9110S #2 VCC
  3. MT3608 OUT- → L9110S #1 GND
  4. MT3608 OUT- → L9110S #2 GND
שני הבקרים מתחברים לאותו מקור חשמל (חיבור מקבילי). השתמש במפצלי חוט לפיצול החשמל.

תרגיל 5.5: ESP32 ל-MPU6050

  1. פין 3.3V של ESP32 → VCC של MPU6050
  2. GND של ESP32 → GND של MPU6050
MPU6050 צריך 3.3V, לא 5V! שימוש ב-5V יזיק לחיישן. ל-ESP32 יש פלט 3.3V מובנה - השתמש בו.

איזה מתח צריך חיישן MPU6050?

תרגיל 5.6: רשימת בדיקת חיווט

✅ עצור! השתמש במולטימטר לוודא:
  • אין קצר בין + ל-GND
  • עם סוללה מחוברת, VIN של ESP32 מראה 5.0V
  • נורית ESP32 נדלקת כשהסוללה מחוברת

  3. חוט אדום מנוע 2 (ימין) → L9110S #1 OUT2A
  4. חוט שחור מנוע 2 (ימין) → L9110S #1 OUT2B

תרגיל 6.3: חבר L9110S #2

  1. חוט אדום מנוע 3 (אחורה) → L9110S #2 OUT1A
  2. חוט שחור מנוע 3 (אחורה) → L9110S #2 OUT1B
  3. חוט אדום מנוע 4 (שמאל) → L9110S #2 OUT2A
  4. חוט שחור מנוע 4 (שמאל) → L9110S #2 OUT2B

⏱️ 25 דקות

תרגיל 9.2: זיהוי סוגי מדחפים

למדחפי CW ו-CCW יש זוויות להב שונות:

  • CW (עם השעון): קצה להב מוביל נוטה ימינה כשמסתכלים מלמעלה
  • CCW (נגד השעון): קצה להב מוביל נוטה שמאלה כשמסתכלים מלמעלה
רוב סטי המדחפים מסומנים או מקודדים בצבעים. בדוק את האריזה!

  4. מנוע שמאל: לחץ מדחף CCW
