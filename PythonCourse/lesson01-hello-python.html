תרגיל 3.1: פעולות חשבון בסיסיות

תרגיל 3.2: כמה שניות יש בשנה?

בואו נחשב! בשנה יש 365 ימים, ביום 24 שעות, בשעה 60 דקות, ובדקה 60 שניות.

seconds_in_year = 365 * 24 * 60 * 60
print("מספר השניות בשנה:")
print(seconds_in_year)
מספר השניות בשנה:
31536000

כמה שניות יש בשנה? (כתבו את המספר)

תרגיל 3.3: סדר פעולות

Python מכבד את סדר פעולות החשבון (כמו במתמטיקה):

print(2 + 3 * 4)      # כפל קודם: 2 + 12 = 14
print((2 + 3) * 4)    # סוגריים קודם: 5 * 4 = 20
print(10 / 2 + 3)     # חילוק קודם: 5.0 + 3 = 8.0
print(10 / (2 + 3))   # סוגריים קודם: 10 / 5 = 2.0

משימה 4: הערות — הודעות לעצמך

⏱️ 10 דקות

תרגיל 4.1: הערות בשורה אחת

הערות הן הודעות שנכתבות בקוד אבל Python מתעלם מהן. הן נועדו בשבילך — להסביר מה הקוד עושה.

# זוהי הערה - Python מתעלם מהשורה הזו
print("שורה זו תרוץ")  # הערה בסוף שורה

# חישוב ממוצע ציונים
grade1 = 90
grade2 = 85
grade3 = 92
average = (grade1 + grade2 + grade3) / 3
print(average)  # מדפיס את הממוצע
שורה זו תרוץ
89.0

תרגיל 4.2: הערות מרובות שורות

כשרוצים לכתוב הערה ארוכה על כמה שורות:

"""
תוכנית מחשבון פשוטה
נכתבה על ידי: [השם שלך]
תאריך: 2026
"""

# אפשר גם ככה:
# שורה ראשונה של הערה
# שורה שנייה של הערה
# שורה שלישית של הערה

print("התוכנית רצה!")

איזה סימן מתחיל הערה בשורה אחת ב-Python?

תרגיל 4.3: למה הערות חשובות?

השוו בין שני קטעי קוד. מי יותר ברור?

# בלי הערות - מבלבל! 😵
x = 1000
y = 0.05
z = x * y * 12
print(z)

# עם הערות - ברור! 😊
salary = 1000            # משכורת חודשית
tax_rate = 0.05          # אחוז מס
annual_tax = salary * tax_rate * 12  # מס שנתי
print(annual_tax)        # מדפיס: 600.0
כלל זהב: כתבו הערות שמסבירות למה הקוד עושה משהו, לא רק מה הוא עושה.
✅ עצור! ודא שאתה יודע: מה ההבדל בין # (שורה אחת) לבין """ (מרובה שורות).
