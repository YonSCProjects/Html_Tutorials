תרגיל 1.1: יצירת משתנים

משתנה הוא כמו קופסה עם שם — אתה שם בתוכה ערך ואחר כך יכול להשתמש בו.

  1. צרו קובץ חדש בשם variables.py
  2. כתבו את הקוד הבא:
name = "יוסי"
age = 14
height = 1.68

print("שם:", name)
print("גיל:", age)
print("גובה:", height)
שם: יוסי
גיל: 14
גובה: 1.68
שימו לב: ב-Python אין צורך להצהיר על סוג המשתנה. Python מבין לבד!

תרגיל 1.3: שינוי ערכי משתנים

score = 0
print("ניקוד התחלתי:", score)

score = score + 10
print("אחרי בונוס:", score)

score = score + 5
print("אחרי עוד נקודות:", score)

# דרך קצרה לעדכון
score += 20
print("ניקוד סופי:", score)

אחרי בונוס: 10
אחרי עוד נקודות: 15
ניקוד סופי: 35

משימה 2: סוגי נתונים

⏱️ 13 דקות

תרגיל 2.1: הכרת הסוגים

ב-Python יש 4 סוגי נתונים בסיסיים:

# str - מחרוזת (טקסט)
name = "Python"

# int - מספר שלם
year = 2026

# float - מספר עשרוני
pi = 3.14159

# bool - ערך אמת/שקר
is_fun = True

print(type(name))     # <class 'str'>
print(type(year))     # <class 'int'>
print(type(pi))       # <class 'float'>
print(type(is_fun))   # <class 'bool'>
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>

מה הפקודה שבודקת את סוג המשתנה?

תרגיל 2.2: בניית משפטים מפרטים אישיים

בואו נשלב משתנים לתוך משפטים באמצעות f-string:

name = "דני"
age = 13
city = "תל אביב"
hobby = "תכנות"

# f-string - הדרך הקלה לשלב משתנים בטקסט
print(f"שלום! שמי {name}")
print(f"אני בן {age} מ{city}")
print(f"התחביב שלי הוא {hobby}")
print(f"בעוד 10 שנים אהיה בן {age + 10}")
שלום! שמי דני
אני בן 13 מתל אביב
התחביב שלי הוא תכנות
בעוד 10 שנים אהיה בן 23
f-string מתחיל באות f לפני הגרשיים, ומשתנים נכנסים בתוך סוגריים מסולסלים {}

תרגיל 2.3: החלפת משתנים

אתגר קלאסי: איך מחליפים ערכים בין שני משתנים?

a = "שוקולד"
b = "וניל"
print(f"לפני: a={a}, b={b}")

# ב-Python זה קל מאוד!
a, b = b, a
print(f"אחרי: a={a}, b={b}")
לפני: a=שוקולד, b=וניל
אחרי: a=וניל, b=שוקולד

🎉 משימה 2 הושלמה!

למדת: str, int, float, bool, הפקודה type(), ו-f-string!

תרגיל 3.1: המרות בסיסיות

לפעמים צריך להמיר ערך מסוג אחד לאחר:

# מחרוזת למספר
age_text = "14"
age_number = int(age_text)
print(age_number + 1)  # 15 - עכשיו אפשר לחשב!

# מספר למחרוזת
score = 100
score_text = str(score)
print("הניקוד שלך: " + score_text)

# מספר שלם לעשרוני
x = float(7)
print(x)  # 7.0

# עשרוני לשלם (חותך את הנקודה!)
y = int(3.99)
print(y)  # 3 (לא 4!)
15
הניקוד שלך: 100
7.0
3

תרגיל 3.2: שנות כלב

אומרים ששנה אנושית שווה 7 שנות כלב. בואו נבנה מחשבון!

human_age = 14
dog_years = human_age * 7

print(f"אם אתה בן {human_age}")
print(f"בשנות כלב אתה בן {dog_years}! 🐕")
אם אתה בן 14
בשנות כלב אתה בן 98! 🐕

כמה שנות כלב יש למי שבן 10? (חשבו: 10 * 7)

תרגיל 3.3: הבעיה הנפוצה — חיבור מחרוזות ומספרים

# זה יגרום לשגיאה! ❌
# print("הגיל שלי: " + 14)

# הפתרון: המרה למחרוזת ✅
print("הגיל שלי: " + str(14))

# או: שימוש ב-f-string (עדיף!) ✅
print(f"הגיל שלי: {14}")

# או: שימוש בפסיק ב-print ✅
print("הגיל שלי:", 14)
✅ עצור! ודא שאתה מכיר: int(), float(), str() — ויודע מתי להשתמש בכל אחת.

תרגיל 4.1: כרטיס מידע אישי

בואו נבנה תוכנית שמציגה כרטיס מידע אישי מפורט:

# פרטים אישיים
name = "שירה"
birth_year = 2012
current_year = 2026
city = "ירושלים"
hobby = "רובוטיקה"
favorite_food = "פיצה"

# חישובים
age = current_year - birth_year
age_in_months = age * 12
age_in_days = age * 365
age_in_hours = age_in_days * 24
age_in_seconds = age_in_hours * 60 * 60

# הדפסת הכרטיס
print("=" * 35)
print("   כרטיס מידע אישי")
print("=" * 35)
print(f"  שם: {name}")
print(f"  עיר: {city}")
print(f"  גיל: {age}")
print(f"  תחביב: {hobby}")
print(f"  אוכל אהוב: {favorite_food}")
print("-" * 35)
print(f"  הגיל שלך בחודשים: {age_in_months}")
print(f"  הגיל שלך בימים: {age_in_days}")
print(f"  הגיל שלך בשעות: {age_in_hours:,}")
print(f"  הגיל שלך בשניות: {age_in_seconds:,}")
print("=" * 35)

אם מישהו נולד בשנת 2010, כמה חודשים חי עד 2026? (חשבו: (2026-2010)*12)

תרגיל 4.2: שנה אותו!

עכשיו שנו את הפרטים לפרטים שלכם!

  1. שנו את name לשם שלכם
  2. שנו את birth_year לשנת הלידה שלכם
  3. שנו את שאר הפרטים
  4. הריצו ובדקו שהחישובים נכונים

תרגיל 4.3: אתגר בונוס — עוד חישובים!

הוסיפו את החישובים הבאים לכרטיס:

# כמה שנים עד גיל 100?
years_to_100 = 100 - age
print(f"  שנים עד גיל 100: {years_to_100}")

# באיזו שנה תהיה בן/בת 100?
year_at_100 = birth_year + 100
print(f"  תהיה בן/בת 100 בשנת: {year_at_100}")

# כמה פעמים הלב פעם (72 פעימות בדקה)
heartbeats = age_in_seconds * 72
print(f"  פעימות לב עד היום: {heartbeats:,}")
✅ עצור! ודא שהכרטיס האישי שלך מציג את כל הנתונים הנכונים עם הפרטים שלך.
