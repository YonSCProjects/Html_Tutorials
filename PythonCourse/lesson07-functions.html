תרגיל 2.3: החזרת מספר ערכים

  1. פונקציה יכולה להחזיר כמה ערכים בו-זמנית:
def min_max(numbers):
    return min(numbers), max(numbers)

smallest, biggest = min_max([4, 7, 1, 9, 3])
print(f"הקטן: {smallest}, הגדול: {biggest}")

הקטן: 1, הגדול: 9
✅ עצור! האם אתה מבין את ההבדל בין print ו-return? print מדפיס למסך, return מחזיר ערך שאפשר לשמור במשתנה.

תרגיל 3.2: *args — מספר לא ידוע של ארגומנטים

  1. *args אוסף את כל הארגומנטים לטאפל:
def average(*numbers):
    if len(numbers) == 0:
        return 0
    return sum(numbers) / len(numbers)

print(average(90, 80, 70))         # 80.0
print(average(100, 95, 88, 72))    # 88.75

מה יחזיר average(10, 20, 30)?

תרגיל 3.3: **kwargs — ארגומנטים עם שמות

  1. **kwargs אוסף ארגומנטים עם שמות למילון:

משימה 4: פונקציות Lambda

⏱️ 8 דקות

תרגיל 4.1: פונקציות אנונימיות בשורה אחת

  1. lambda יוצרת פונקציה קצרה בשורה אחת:
# פונקציה רגילה
def double(x):
    return x * 2

# אותו דבר עם lambda
double = lambda x: x * 2

print(double(5))  # 10

תרגיל 4.2: Lambda עם sorted, filter, map

  1. מיון רשימה לפי קריטריון מותאם:
students = [("דני", 85), ("מיכל", 92), ("אורי", 78)]

# מיון לפי ציון (מהגבוה לנמוך)
sorted_students = sorted(students, key=lambda s: s[1], reverse=True)
print(sorted_students)

# סינון — רק מי שעבר 80
passed = list(filter(lambda s: s[1] >= 80, students))
print(passed)

# מיפוי — הוספת 5 נקודות בונוס לכולם
bonus = list(map(lambda s: (s[0], s[1] + 5), students))
print(bonus)

[('מיכל', 92), ('דני', 85), ('אורי', 78)]
[('דני', 85), ('מיכל', 92)]
[('דני', 90), ('מיכל', 97), ('אורי', 83)]
✅ עצור! Lambda שימושי למיונים וסינונים מהירים. לפונקציות מורכבות — השתמש ב-def רגיל.

משימה 5: Scope — טווח משתנים

⏱️ 8 דקות

תרגיל 5.1: Local vs Global

  1. משתנים בתוך פונקציה הם מקומיים (local):
x = 10  # משתנה גלובלי

def my_func():
    x = 5  # משתנה מקומי — לא משנה את הגלובלי!
    print(f"בתוך הפונקציה: x = {x}")

my_func()
print(f"בחוץ: x = {x}")

בתוך הפונקציה: x = 5
בחוץ: x = 10

מה הערך של x בחוץ אחרי הקריאה לפונקציה?

תרגיל 5.2: כלל LEGB

LEGB Rule: Python מחפש משתנים בסדר הזה: Local (בתוך הפונקציה) → Enclosing (בפונקציה עוטפת) → Global (ברמת הקובץ) → Built-in (מובנה ב-Python) 
name = "גלובלי"  # G

def outer():
    name = "חיצוני"  # E

    def inner():
        name = "פנימי"  # L
        print(name)

    inner()

outer()  # מדפיס: פנימי

תרגיל 6.1: ממיר טמפרטורה

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

def fahrenheit_to_celsius(f):
    return (f - 32) * 5/9

# בדיקה

print(celsius_to_fahrenheit(100))  # 212.0
print(fahrenheit_to_celsius(98.6)) # 37.0

תרגיל 6.2: מחשבון BMI

def calculate_bmi(weight_kg, height_m):
    bmi = weight_kg / (height_m ** 2)
    if bmi < 18.5:
        category = "תת משקל"
    elif bmi < 25:
        category = "משקל תקין"
    elif bmi < 30:
        category = "עודף משקל"
    else:
        category = "השמנה"
    return round(bmi, 1), category

bmi, cat = calculate_bmi(70, 1.75)
print(f"BMI: {bmi} — {cat}")

תרגיל 6.3: בודק חוזק סיסמה

def check_password(password):
    score = 0
    if len(password) >= 8:
        score += 1
    if any(c.isupper() for c in password):
        score += 1
    if any(c.islower() for c in password):
        score += 1
    if any(c.isdigit() for c in password):
        score += 1
    if any(c in "!@#$%^&*" for c in password):
        score += 1

    levels = ["חלשה מאוד", "חלשה", "בינונית", "חזקה", "חזקה מאוד", "מעולה!"]
    return score, levels[score]

score, level = check_password("MyP@ss1")
print(f"ציון: {score}/5 — {level}")

תרגיל 6.4: בודק מספר ראשוני ומטיל קוביה

import random

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def roll_dice(sides=6):
    return random.randint(1, sides)

# בדיקות
print(is_prime(7))    # True
print(is_prime(10))   # False
print(f"הקוביה הראתה: {roll_dice()}")
print(f"קוביה של D20: {roll_dice(20)}")
