תרגיל 1.2: range() — הכלי החשוב ביותר

range() יוצר רצף של מספרים. יש לו 3 צורות:

# צורה 1: range(stop) — מ-0 עד stop-1
for i in range(5):
    print(i)   # ידפיס: 0, 1, 2, 3, 4

# צורה 2: range(start, stop)
for i in range(1, 6):
    print(i)   # ידפיס: 1, 2, 3, 4, 5

# צורה 3: range(start, stop, step)
for i in range(0, 10, 2):
    print(i)   # ידפיס: 0, 2, 4, 6, 8

מה ידפיס range(3, 12, 3)? (רשמו את המספרים מופרדים בפסיקים)

תרגיל 4.3: אתגר FizzBuzz!

        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")

תרגיל 5.1: בניית המשחק

המחשב יבחר מספר אקראי והשחקן ינחש! המשחק ייתן רמזים "גבוה מדי" או "נמוך מדי".

  1. צרו קובץ guessing_game.py
  2. כתבו את הקוד הבא:
import random

secret = random.randint(1, 100)
attempts = 0

print("🎲 ברוכים הבאים למשחק ניחוש המספרים!")
print("בחרתי מספר בין 1 ל-100. נסו לנחש!")

while True:
    guess = int(input("הניחוש שלך: "))
    attempts += 1

    if guess < secret:
        print("📈 נמוך מדי! נסו מספר גבוה יותר")
    elif guess > secret:
        print("📉 גבוה מדי! נסו מספר נמוך יותר")
    else:
        print(f"🎉 מצאת! המספר היה {secret}")
        print(f"לקח לך {attempts} ניסיונות")
        break

תרגיל 5.2: שדרגו את המשחק!

הוסיפו מגבלת ניסיונות — השחקן מקבל 7 ניסיונות בלבד:

import random

secret = random.randint(1, 100)
max_attempts = 7

print("🎲 יש לך 7 ניסיונות לנחש מספר בין 1-100!")

for attempt in range(1, max_attempts + 1):
    guess = int(input(f"ניסיון {attempt}/{max_attempts}: "))

    if guess < secret:
        print("📈 נמוך מדי!")
    elif guess > secret:
        print("📉 גבוה מדי!")
    else:
        print(f"🎉 ניצחת בניסיון {attempt}!")
        break
else:
    print(f"😢 הפסדת! המספר היה {secret}")
שימו לב ל-else אחרי for! הוא רץ רק אם הלולאה סיימה בלי break.

כמה ניסיונות מינימום נדרשים כדי לנחש מספר בין 1-100 בשיטת "חיפוש בינארי" (חצי-חצי)?
